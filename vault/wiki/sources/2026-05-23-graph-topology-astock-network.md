# A股相关性网络：图论 / 拓扑结构选股（科普）

> 来源：[微信公众号](https://mp.weixin.qq.com/s/P_zpSMzI7qwB9TiE442U6A) | 2026-05-23 自动提取失败，用户手动复制粘贴 ingest

## 一句话定位

**科普级**（无公式 / 无参数 / 无回测 / 无绩效）：把全市场当成一张**相关性网络**——节点=个股、边=价格相关性，用**最小生成树(MST)** 剪枝留强连接、算**中心度(Centrality)** 找枢纽，再用枢纽异动预判**结构性传染暴跌**做风控。是 [[截面思维]] 的"网络/图论"叙事版，与 [[低相关组合]]、[[ETF候选池构建]]（聚类去重）同源。

## 核心链条（文章逻辑）

```
全市场 5000 股
   ↓  价格变动相关性矩阵（Correlation Matrix）
相关性图：节点=股、边=相关（粗细=相关强度）
   ↓  最小生成树 MST 剪枝（割弱边，留最强"资金脐带"）
"A股地形图"：识别 孤儿节点 / 枢纽节点（中心度高）
   ↓
用法① 枢纽异动 → 预判资金流向 / 结构性雪崩 → 风控清仓（文章主线）
用法② 找"跨两个赛道的桥梁标的" → 轮动里涨最快（选股）
用法③ 查个股"朋友圈"危险邻居 → 避雷（散户避坑版）
```

## 风险与待验证（诚实标注）

- **AMBIGUOUS / UNVERIFIED**：纯科普，**无任何公式、参数、回测、绩效数字**；"0.001 秒预判""10 秒感染全网 30%"等是修辞，非可验证指标。

## ETF 7.0 项目集成结论（实测对接）

把文章拆成三种用法，对照 ETF 7.0 已确立的结论：

| 用法 | 本质 | 判断 |
|---|---|---|
| 枢纽异动→传染风控→清仓（主线） | 市场择时 / regime overlay | **先验强负**：ER gate、纯 RSRS、RSRS+MA20+量能 已在 ETF 池**4 次否决**；择时砍反弹、降不了已 −18.5% 的回撤。 |
| 中心度当选股因子 | 截面 alpha | 弱：114 ETF 上中心度≈高 beta 代理，且与突破腿高相关；文章零证据。 |
| **相关性图做"聚类上限"分散约束** | **风险约束（非择时非 alpha）** | **唯一契合**：滚动相关性图→连通分量聚类→约束突破腿"同簇 ≤N 只"，降同簇共振回撤。对齐 [[低相关组合]] 与 ETF2.0"类别上限有效"老结论。已建 PoC（`ETF 7.0/cluster_cap_test.py`）跑 IS/OOS 对照。 |

> 关键提炼：**借它的"工具"（相关性图聚类），弃它的"主线用法"（拓扑择时）。**
