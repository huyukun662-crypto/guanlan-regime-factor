# 每日一个金融指标：SAR（抛物线转向指标）

> 来源：[聚宽社区](https://www.joinquant.com/view/community/detail/04f34d63055c3ddd06c406f4e7282c66?type=1) | "每日一个金融指标"系列 | 2026-05-06 通过用户手动复制粘贴 ingest

## 一句话定位

科普向介绍 [[SAR指标]]（Stop and Reverse / 抛物线转向）— 由威尔斯·威尔德发明的经典技术分析指标，**用抛物线轨迹跟踪趋势 + 提示反转点 + 设置动态止损**。SAR 点在 K 线下方 = 多头持仓；上方 = 空头持仓；价格穿透 SAR 点 = 反转信号。

## 核心公式

**上涨趋势 SAR**（绿点）：
```
SAR_n = SAR_{n-1} + AF × (EP - SAR_{n-1})
```

**下跌趋势 SAR**（红点）：
```
SAR_n = SAR_{n-1} - AF × (SAR_{n-1} - EP)
```

- **AF（加速因子）**：初始 0.02，每次创新高/新低 +0.02，上限 0.2
- **EP（极值点）**：当前趋势内的最高价（上涨）或最低价（下跌）

## 三大核心用法

### 1. 趋势跟踪持仓信号
- SAR 点在 K 线**下方** → 多头趋势 → 持有/加仓
- SAR 点在 K 线**上方** → 空头趋势 → 空仓/做空
- 点距扩大 → 趋势加速；点距走平 → 趋势衰竭

### 2. 动态止损与止盈
- 价格跌破 SAR 点 → 止损（任何持仓周期）
- 短线：SAR 点下方 1% 作为保护性止盈
- 长线：结合周线 SAR 锁定大利润

### 3. 趋势反转信号
- 多转空：股价从上方跌破 SAR → 卖出 + 反手做空
- 空转多：股价从下方升破 SAR → 买入 + 反手做多

## 实战增强技巧

### 参数优化（不同行情）
| 行情 | AF 初始 | AF 上限 | 搭配指标 |
|---|---|---|---|
| 短线波段 | 0.04 | 0.2 | MACD(5,13,1) |
| 中线趋势 | 0.02 | 0.1 | 60 日均线 |
| 极端行情 | 0.2（固定）| 0.2 | ATR(14) |

### 多周期共振
- 日线 + 周线 SAR 同向 → 大级别趋势确认
- 小时线 SAR 转向 → 短期变盘预警

## 三大优势 / 三大陷阱

| 优势 | 陷阱 |
|---|---|
| 傻瓜式操作（红绿点清晰）| 震荡市频繁转向（需 RSI 配合）|
| 动态风控（自动上移止损）| 加速因子滞后（初期可能错过）|
| 单边行情捕捉主升 / 主跌浪 | 缺口跳空失灵（穿透 SAR）|

## 与已有素材的关联

### 经典技术分析家族（同主题）
本指标与已有的 [[均线偏离度]] / [[RSRS]] / [[MACD]] / [[KDJ]] / [[鳄鱼线]] 同属 [[技术指标择时]] 主题。SAR 的独特性：
- vs [[均线偏离度]]：均线偏离度只给"强弱信号"，SAR **同时给止损位**
- vs [[RSRS]]：RSRS 是基于高低价回归的相对强度；SAR 是基于历史极值的固定追赶
- vs [[MACD]]：MACD 是趋势确认；SAR 是**趋势 + 止损**双功能
- vs [[KDJ]]：KDJ 是震荡指标；SAR 是趋势指标（震荡市反而失效）
- vs [[鳄鱼线]]：鳄鱼线用三均线判断状态；SAR 用单一抛物线 + 加速因子

### 与浙商证券系列二的关联
[[2025-07-31-zheshang-rate-timing-winrate]] 中提到 K 线 / MACD / KDJ 三类经典指标 — SAR 是**第四类经典指标**（趋势 + 止损一体），可加入 [[多信号策略]] 作为额外子模块。
