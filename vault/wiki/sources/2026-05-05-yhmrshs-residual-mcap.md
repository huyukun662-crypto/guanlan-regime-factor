# 特异市值因子

> 来源：[微信公众号 量化拯救散户](https://mp.weixin.qq.com/s/1H6uYmkTJKvtmP_-YE_qiA) | 2026-05 | 个人量化博主

## 一句话定位

**特异市值（Residual Market Cap）** = 用**对数市值**对**行业哑变量 + 对数净资产 + 资产负债率 + 对数正净利润 + 对数负净利润绝对值**做截面回归后的**残差**。代表**市值无法被基本面解释的部分**，衡量"内生市值"的偏离度 — 因子值大 → 市值高估 → 未来收益更可能下跌（**反向因子**）。

## 学术源头

老外 **2005 年论文**首次提出（具体作者文章未明示）。**21 年来在国内研报中几乎未见介绍**。

> 作者复盘前的预期："国内表现应该不好" → 实际复现后发现"分层回测结果让我眼前一亮，甚至比计算超级复杂的量价因子还好"

## 计算步骤

### 截面回归模型

```
ln(market_cap) = α + β₁·industry_dummies + β₂·ln(net_assets)
              + β₃·leverage + β₄·ln(净利润正部分) + β₅·ln(|净利润负部分|) + ε
```

**特异市值 = ε**（残差）

### 关键变量定义

| 变量 | 计算 |
|---|---|
| 对数市值 | `ln(market_cap)` |
| 对数净资产 | `ln(total_assets - total_liability)` |
| 资产负债率 | `total_liability / total_assets` |
| 净利润正部分 | `ln(max(net_profit, 0))` |
| 净利润负部分 | `ln(\|min(net_profit, 0)\|)` |
| 行业哑变量 | 申万一级行业（sw_l1）one-hot |

### 数据频率
- 行业 / 市值：日频（market_cap 每日变化）
- 财报数据：单季度（lag=0Q balance / lag=1Q income）

### Python 代码核心
```python
def cal_factors(self, i):
    y = log_market_cap[i]                         # 被解释变量
    x = concat([ind_dummies, ln_net_assets,
                leverage, ln_profit_plus,
                ln_profit_minus_abs])              # 解释变量
    flag = ~np.isnan(y)
    if (flag.sum()) < 100:
        return NaN  # 截面太小则跳过
    x_data = np.nan_to_num(x[:, flag], posinf=0, neginf=0)
    lr = LinearRegression().fit(x_data.T, y[flag])
    residual = y[flag] - lr.predict(x_data.T)
    return Series(residual, index=codes[flag])
```

并行化：`joblib.Parallel(n_jobs=16)` 跨交易日并行计算。

## 因子逻辑（重要 — 经济直觉）

> "特异市值代表了市值不可被公司基本面或市场解释的部分，衡量了当前市值与**内生市值**的偏离程度。
> 因子值越大 → 市值向上偏离内生市值越多 → **市值被高估** → 未来股票收益率越有可能下跌"

⇒ **本质是基本面"应有市值"的均值回归**：
- 用回归找出"基于基本面合理的市值"
- 残差 > 0 → 市场给的市值高于基本面应有 → 反转下跌概率高
- 残差 < 0 → 市值被低估 → 未来反转上涨概率高

## 因子评价

### 低频化方法（重要细节）
作者发现**用过去 21 个交易日的标准差**做低频化（而非均值）→ IC 表现更好。
- ⇒ **波动率版本** 优于 **均值版本**

### IC 表现
- 中规中矩
- 3 年 IC 绝对值 > 0.08
- 2 年 IC 绝对值 < 0.06

### 收益分析（亮点）
- **完美单调性**：分层回测在整个区间表现单调
- **均值低频化版本表现略差**（验证 21 日 std 优于均值的假设）

### 已自带行业 / 市值中性化
因为回归中已包含行业哑变量 + 市值（被解释变量），**因子评价时不再做二次中性化**。

## 与已有素材的关联

### 与同期行为金融因子的对比
- [[凸显性因子STR]]（招商 / 方大 2022）：极端收益扭曲决策权重
- [[球队硬币因子]]（方正 2022）：体育博彩动量迁移
- **本因子（特异市值）**：基本面残差思路 — 与 STR / 球队硬币**机制完全不同**，但同属"低相关性"另类 alpha
- ⇒ 三个因子可能**正交性较好**，适合合成

### 与"老外因子在 A 股失效"的反例
- [[2026-05-05-qpb-industry-pricevolume-etf-rotation]] 给出反例：华西因子在 ETF 上失效
- 本因子是个**反向反例**：21 年前的老外因子 + 国内研报无介绍 + 作者预期不高 → **反而在 A 股有效**
- ⇒ 启示：**不能仅凭"国内是否有研报"判断因子在 A 股的有效性**，需亲自跑
