# 如何区分主线是调整还是终结？

> 来源：[广发策略公众号](https://mp.weixin.qq.com/s/7HxIZWMiUpbVl7RrgaH01g) | [[刘晨明]] / 郑恺 / 杨清源 | 2025-09-14

## 一句话定位

**[[均线偏离度]] 指标的原创报告**（除法版）— 复盘 2012 年以来主线行情的乖离率分组绩效，给出量化阈值用于判断主线行情的"过热 / 良性 / 转弱"，本文是 [[2026-05-05-gf-strategy-residential-entry-thermometer|6 大指标看居民入市]] 中减法版的前置版本。

## 核心观点

### 一、"量顶"≠ 趋势结束（保持牛市思维）
- 复盘 2010 年以来全 A "量顶"后的表现 — 多为加速结束 + 趋势放缓，**而非趋势终结**
- 资金面"四个蓄水池"（监管资金 / 保险等绝对收益 / 居民部门 / 海外）尚未挑战流动性牛市叙事

### 二、均线偏离度公式（原创版 / 除法版）

```
均线偏离度 = (对数收盘价 / 20日EMA) - 1     # 除法版，原始报告中的形式
```

**为什么用 EMA20 而非 MA20**：EMA 给近期数据更高权重，趋势行情中收盘价与 EMA 更贴合（参考阿尔·布鲁克斯《高级趋势技术分析》）。

**对偏离度的"双重过滤"**：
1. 阈值过滤（见下方）
2. 趋势过滤：最近 20 日中收盘价至少 10 日在均线上方，否则已表明趋势转弱

### 三、阈值（除法版，行业指数标定）

**入场**：
- 偏离度 > 2% 容易吃到回调（追高代价大）
- 偏离度 0.6%-1.8% 适中入场（最优区间）
- 偏离度 < 0.6%/为负 可能面临趋势转弱

**离场**：
- 均线上方无需担忧
- 偏离度 -0.6%~0%（刚跌穿均线）建议坚守，T+20/40/60 平均收益 0.64%/6.15%/8.22%
- 偏离度 < -0.6% 危险信号 → 离场
- "20 日内 10 日收线下" 即使偏离度不极端 → 缓慢失去趋势 → 离场

### 四、典型主线行情样本（2012-2025）

| 时段 | 典型主线 |
|---|---|
| 2012-2015 | TMT、创业板、一带一路、券商 |
| 2019-2021 | 半导体、茅指数、新能源车、创新药、光伏 |
| 2023-2025 | AI 产业链（光模块、PCB、机器人）、国产算力（科创芯片） |

### 五、本轮主线（2025/9/12）读数（除法版口径）
- 光模块 1.3% / PCB 2.0% / 创新药 0.1% / 科创芯片 1.2%
- 当前主线均未释放止损信号
