# QuantsPlaybook — 100+ 量化策略 Python 复现仓库

> 来源：[github.com/hugo2046/QuantsPlaybook](https://github.com/hugo2046/QuantsPlaybook) | [[hugo2046]] | master 分支 | 最近更新 2025-12-14

## 一句话定位

**国内中文量化界最完整的开源策略复现仓库之一** — 系统性复现 100+ 国内券商金工研报的 Python 实现（Jupyter Notebook），按"择时 / 因子 / 价值 / 组合优化"四大类组织。**与本知识库的研报型素材完全对应**，可作为"理论 → 代码"的桥梁。

## 仓库结构

```
QuantsPlaybook/
├── A-量化基本面/      # 价值投资策略 (2 个)
├── B-因子构建类/      # 多因子模型 (22+ 个)
├── C-择时类/         # 市场择时 (25+ 个)
├── D-组合优化/       # 投资组合 (2 个)
├── hugos_toolkit/    # 通用工具库
└── SignalMaker/      # 择时信号生成器
```

## 复现质量自评（README 自述）

- **平均年化收益**（按类别）：择时 12.8% / 因子 15.2% / 组合 10.5%
- **最大回撤**：择时 18.5% / 因子 22.3% / 组合 15.8%
- **夏普比率**：择时 0.85 / 因子 0.92 / 组合 0.78
- **胜率**：择时 58% / 因子 62% / 组合 55%

⚠ 这是**仓库整体平均**，不代表每个策略。具体策略需各自阅读 ipynb 验证。

## V25 项目集成路径

1. **Quick win**：先 `git clone` 仓库到 V25 项目作为参考
2. **优先复现**（已在知识库中有理论锚点的策略）：
   - `C-择时类/RSRS择时指标/` → 与 [[2026-05-05-openalphas-rsrs-market-thermometer]] 对照实现
   - `C-择时类/低延迟趋势线与交易择时/` → 与浙商系列二中的[[低延时趋势]]对照
   - `C-择时类/指数高阶矩择时/` → 与浙商系列二中的[[阶矩择时]]对照
   - `C-择时类/均线交叉结合通道突破择时研究/` → 与浙商系列二中的[[通道过滤]]对照
3. **新增方向**（仓库已有但知识库缺少理论锚点）：
   - **HHT 模型**（招商 2024，改进希尔伯特-黄变换）
   - **球队硬币因子**（方正 2022，行为金融）
   - **凸显性因子 STR**（招商 / 方大 2022）
   - **特征分布建模择时**（华创 2022 龙虎榜模型）
4. **数据源切换**：仓库默认 JQData / Tushare → V25 已用 efinance / akshare，需改写数据接口

## 风险提示

- 仓库代码主要是**研报复现**，不一定经过严格 OOS 验证，回测过拟合风险存在（这也是仓库自身收录"CSVC 框架"和"过拟合概率检验"的原因之一）
- 部分链接指向聚宽社区或 NBViewer，**克隆仓库本地无完整 ipynb**
- 数据依赖第三方 API → 长期可用性受限
