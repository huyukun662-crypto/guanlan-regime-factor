# RSRS 把高低价通道做成胜率 68%+ 夏普 1.8+ 的市场温度计

> 来源：[小红书 @OpenAlphas](https://www.xiaohongshu.com/discovery/item/69dc71ba0000000021011deb) | 23 赞 | A 档（公式完整 + 已有学术先例）

- **作者**：[[OpenAlphas]]（RSRS 原创非作者，是光大金工 2017 年提出）

- **策略类型**：技术指标择时（指数级别）
- **标的**：指数级别（不适合个股）

## 核心观点

### 1. 传统阻力支撑分析的痛点
- 突破信号滞后
- 震荡市反复打脸
- 大熊市支撑位逐级失效
- **根源**：把阻力支撑当作固定阈值，忽略市场动态

### 2. RSRS 核心思想
- 不再关注"具体点位"，转向"**相对强度**"变化
- 用每日最高价 / 最低价 OLS 回归得到 β 斜率：
  ```
  high_t = α + β · low_t + ε_t,   ε_t ~ N(0, σ²)
  ```
- β > 1：支撑强度 > 阻力强度（看多）
- β < 1：阻力强度 > 支撑强度（看空）

### 3. 三层标准化（解决固定阈值问题）
1. **标准分**：z_t = (β_t - μ_β) / σ_β
2. **修正标准分**：z_t · R_t²（拟合可靠性加权）
3. **右偏标准分**：z_t · R_t² · β_t = 相对位置 × 拟合可靠性 × 绝对强度

### 4. 信号语义（牛/震荡/熊三态）
| 状态 | 支撑>阻力 | 阻力>支撑 |
|---|---|---|
| 牛市 | 持续上涨 | 即将结束 |
| 震荡 | 牛市启动 | 熊市启动 |
| 熊市 | 即将见底 | 加速下跌 |

### 5. 解决熊市抄底问题（左侧交易倾向）
- **价格趋势过滤（4.1）**：MA20 上升趋势（当日 MA20 > 3 日前 MA20）才接受买入信号
  - 效果：MDD 从 50%+ 降低到 20%
- **成交量相关性过滤（4.2）**：10 日成交量与 RSRS 修正标准分正相关时才买入
  - 效果：胜率 + 盈亏比提升，收益略逊于 4.1
