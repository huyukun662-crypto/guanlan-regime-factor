# 浙商证券：中国市场可转债定价模型比较研究与实战应用

> 在转债市场"高估高价、超买生态"下，BS 模型作进攻型定价锚、ZL 模型作防守型定价锚，错误定价因子是与传统五因子高度正交的另类 Alpha。

## 核心观点

1. **绝对定价框架替代失效的相对估值**：在转债估值中枢全面抬升、平价溢价率等相对估值指标被情绪与拥挤交易扭曲的环境下，引入 [[BS模型]] 与 [[ZL模型]] 两类绝对定价框架，构建统一的可量化坐标系，将"错误定价"提纯为可交易的 Alpha 信号。

2. **两类模型分化定位**：[[BS模型]] 是"进攻型定价锚"——对正股趋势与波动率响应敏锐，因缺失 [[强赎条款]] 天花板约束在牛市系统性高估；[[ZL模型]] 是"防守型定价锚"——基于 [[蒙特卡洛模拟]] 显式处理 [[下修条款]] / [[回售条款]] / [[强赎条款]] 博弈，估值偏保守但在熊市提供扎实债底支撑。

3. **强约束下长周期回测**（2019 年至今，AA- 及以上评级、流动性 + 期限 + 余额硬约束）：
   - BS 多头年化 **19.30%**，最大回撤 22.26%，夏普 1.35
   - ZL 多头年化 **15.33%**，最大回撤 **13.74%**（低于基准 14.91%），夏普 **1.44**（高于 BS）
   - 两模型的纯空头组合年化均为负、回撤接近或超过 30%，证实**严约束下空头逻辑被屏蔽**，定价博弈核心应聚焦多头端。

4. **错误定价因子是另类 Alpha，与传统五因子高度正交**：BS / ZL 偏差因子与流动性、波动率、量价、估值、动量五因子相关性绝对值普遍 ≤ 0.25。在 0.06% 双边交易成本下：
   - **BS 偏差因子**单因子年化超额 11.50%、夏普 1.24；六因子等权合成把组合年化超额推至 **13.92%**（是绝对收益上限的核心利器）
   - **ZL 偏差因子**单因子年化超额 7.19%、夏普 **1.29**（全组最优）、回撤压缩至 13.15%（是组合减震器）
   - 在双层分组 [[风格中性约束]] 下，两类因子在每个风格切片内仍输出高夏普（ZL 在动量中性下夏普 1.63），证实其 Alpha 来自**纯粹期权价值回归**而非隐性风格暴露。

5. **下行周期切换至 ZL，验证空头威力**：放宽至全市场（仅保留剩余期限 + 上市时间约束）、聚焦 2022Q1–2024Q3 单边下行周期：
   - ZL 模型纯空头年化 **14.78%**、夏普 0.88，显著优于 BS 空头（年化 8.52%、夏普 0.66）
   - ZL 多空组合年化 11.49%、夏普 0.90，全面压制 BS 多空（3.93% / 0.47）
   - **状态切换配置范式**：上行 + 估值扩张期重仓 BS 多头；下行 + 信用风险频发期切换至 ZL（多空兼顾）。

6. **后续优化方向**——从静态截面定价转向动态概率调整的 [[事件驱动策略]]：(1) 强赎概率调整捕捉"延迟强赎溢价"；(2) 在 ZL 贴现路径前置接入 Logistic / 神经网络预判下修概率；(3) 结合大股东增持 / 回购等事件信号重估债底，强化空头端平仓指标。

## 关键概念

- [[BS模型]] — 静态欧式期权解析定价，进攻型
- [[ZL模型]] — 蒙特卡洛模拟 + 条款博弈最优停时，防守型
- [[可转债]] — 兼具债性底仓与股性期权的混合证券
- [[错误定价因子]] — 模型理论价格与市场价格的相对偏差（RD），按月滚动调仓
- [[蒙特卡洛模拟]] — ZL 模型实现机制，含 GPU 加速 + 路径依赖处理
- [[强赎条款]] — 期权价值天花板，BS 模型缺失刻画的核心
- [[下修条款]] — A 股转债最具爆发力的事件驱动超额来源
- [[回售条款]] — 投资者侧的流动性硬底
- [[在值程度]] — Moneyness = ln(S/X)，定价偏离的关键解释变量
- [[中证转债指数]] — 回测基准（000832.CSI）
- [[双层分组分析]] — Double-Sort，因子纯度提纯检验方法
- [[风格中性约束]] — 控制传统五因子后再切分定价偏差，验证 Alpha 纯度
- [[事件驱动策略]] — 后续优化方向，引入条款博弈概率调整

## 原文精彩摘录

> 引入具备严密数理支撑的可转债绝对定价模型（BS 与 ZL 模型），成功构建了统一的绝对估值尺度。在传统相对估值体系面临"定价锚缺失"的困局下，该框架旨在系统性剥离情绪溢价，将深挖出的错误定价转化为可验证、可落地且符合投资标准的阿尔法交易信号。

> BS 模型对正股趋势与波动率响应极度敏锐，但因缺乏强制赎回条款的天花板约束，在整体上行的多头市场中容易顺应情绪给出系统性高估的偏离，从而构筑了捕捉股性溢价的高弹性"进攻型定价锚"。

> ZL 模型严格施加了信用利差贴现并自动剔除低评级标的的非理性投机溢价，其估值体系精确且天然偏向保守，在风险偏好回落的熊市环境中能够为组合提供极其扎实的债底支撑与回撤控制。

> 在风险偏好收缩的环境中，ZL 模型的多空组合实现了 11.49% 的年化收益与 0.90 的夏普比率，对 BS 模型多空组合（年化收益 3.93%，夏普 0.47）形成了全面压制。

## 风险提示（原文）

1. 历史回测业绩无法代表或承诺未来真实盘面的投资收益
2. 定价模型先验假设失效与极端尾部风险
3. 政策与市场交易规则非预期变动风险
