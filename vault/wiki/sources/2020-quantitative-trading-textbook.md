# 量化交易（教科书）

> Xin Guo / Tze Leung Lai / Howard Shek / Samuel P. S. Wong | Chapman & Hall 2017 / 高教社 2020 译本 | 355 页 | 8 章 + 2 附录

## 一句话定位

斯坦福 + UC Berkeley 学者 + 业界从业者四人合著的**研究生级量化交易教科书**，定位于**弥合学术研究与金融实务的差距**。是知识库中**首本系统覆盖高频 / 市场微观结构 / 最优执行 / 做市**的学术级素材，与已有研报体形成"理论 vs 实战"的对照。

## 章节地图（B3 详细版）

### 第一章：概论（pp.1-15）
**核心**：量化交易历史回顾 + 学科地图。
- **1.1 交易基础结构演变**：口头交流 → 手势交流 → 电子交易平台
- **1.2 量化交易策略与时间跨度**：基本面（天/周）vs 高频（秒）
- **1.3 统计套利与有效市场假说争论**：EMH 反例
- **1.4 量化基金 / 公募 / 对冲基金分类与表现**
- **1.5 数据 / 分析 / 模型 / 优化 / 算法五维度**
- **1.6 跨学科性**（统计、CS、金融、数学、法律）

### 第二章：量化交易中的统计模型与方法（pp.16-59）
**核心**：从随机游走到鞅回归 + 投资组合理论的不确定性处理。
- **2.1 价格特征**：低频时序分析 + 高频离散化价格变动
- **2.2 巴黎证交所布朗运动 vs 华尔街随机游走**（学术史）
- **2.3-2.4 现代投资组合理论**（Markowitz）+ 统计学基础
- **2.5 [[NPEB法则]]**（Nonparametric Empirical Bayes）— **本团队特色** — 处理参数不确定性的最优权重计算（含 Bootstrap 估计）
- **2.6 随机游走 → ARIMA / GARCH / 鞅回归**演进
- **2.7 新现代投资组合理论**：在 NPEB 中加入时序效应 + 信息比率最大化
- **2.8 统计套利**：非参数回归 + 变点模型 + 动量 / 反转策略

### 第三章：积极型投资组合管理与投资策略（pp.60-96）
**核心**：从被动到主动的投资组合管理。具体小节未深入抓取。

### 第四章：电子交易中的计量经济学（pp.97-136）
**核心**：高频数据的统计建模 — 噪声 / 跳跃 / 积分方差。
- **4.1 电子交易和交易数据**
- **4.2 Roll 模型**（买卖反弹 bid-ask bounce）+ **市场微观结构噪声**模型
- **4.3 积分方差估计**（6 种方法）：
  - 稀疏抽样 / 子样本平均 / 两时间尺度（TSRV）/ 已实现核估计 / 预平均 / QMLE
- 主要解决"高频数据中真实波动率如何剥离微观结构噪声"的问题

### 第五章：限价订单簿（LOB）数据分析与动态模型（pp.137-188）
**核心**：本书最重头的章节 — [[限价订单簿LOB]] 完整建模。
- **5.1 LOB 简介**：限价 vs 市场指令、价差、流动性
- **5.2 LOB 特征**：价格调整机制 + 交易量不平衡（**关键预测信号**）+ 其他指示量
- **5.3 多元点过程拟合 LOB**：用 Hawkes 过程等点过程模型描述订单到达
- **5.4 机器学习分析 LOB**：现代方向
- **5.5 排队论建模**：扩散极限 / 流动极限 / 序列反应模型
- 关键贡献：把 LOB 从"业界经验"变成"数学可建模对象"

### 第六章：最优执行与配置（pp.189-208）
**核心**：[[最优执行]] 完整框架 — 大订单如何拆分。
- **6.1 单资产最优执行**：
  - **Bertsimas-Lo 1998**（开创性论文）
  - **[[Almgren-Chriss模型]] 2001**（实操事实标准）
  - 6.1.3 确定性最优策略
- **6.2 价格乘数影响 + 奇异随机控制**：Guo-Zervos 2015（本作者团队工作）
- **6.3 基于 LOB 数据的供求推断**：Obizhaeva-Wang 2013 / Alfonsi 等 2010
- **6.4 投资组合最优执行**

### 第七章：做市和智能订单路径（pp.209-249）
**核心**：[[做市策略]] + 暗池 / SOR。
- **7.1 [[Ho-Stoll模型]]** + **[[Avellaneda-Stoikov模型]]**：经典做市策略
- **7.2 [[HJB方程]] 求解**及拓展（Hamilton-Jacobi-Bellman 方程）
- **7.3 脉冲控制 + 限制 / 市场订单**
- **7.4 智能订单路径（SOR）+ 暗池**
- **7.5 多交易所最优订单分拆**：成本函数 + 随机逼近方法
- **7.6 暗池删失探索 + 贪心配置算法**

### 第八章：信息学、监管与风险管理（pp.250-291）
**核心**：合规 + 风控。
- **8.1-8.3 自动交易系统的策略 + 交易组成**
- **8.4 交易所规章**：SIP / NMS / 卖空规则 / 熔断机制 / 市场操纵
- **8.5 风险管理**：操作风险 + 策略风险

### 附录 A：鞅理论（pp.292-299）
**核心**：离散 + 连续时间鞅、二次变化、半鞅、随机积分（数学预备）。

### 附录 B：马尔可夫链（pp.300-313）
**核心**：转移概率、平稳分布、遍历性（数学预备）。

### 关键启示

1. **填补理论空白**：知识库已有的"5 子策略 + 多信号"（[[2025-07-31-zheshang-rate-timing-winrate]] 浙商）没有理论基础（更像工程合集）；本书提供了**随机控制论的统一框架**
2. **填补主题空白**：知识库**完全没有 LOB / 最优执行 / 做市**素材 — 本书一次性补齐
3. **填补频率空白**：[[2026-05-05-qpb-etf-intraday-momentum]] 是分钟级 → 本书第 4-5 章可深化到 tick 级
4. **学术追溯**：[[特异市值因子]] 复现的"老外 2005 论文" → 本书提供了同期论文的引用网络（Lai-Xing 2008 等）
5. **空白警示**：本书**不讨论 IS/OOS 工程纪律**（这是 V25 项目复盘 + OpenAlphas 系列的强项）— 复现本书算法时需自己加纪律

## 风险提示

- **2017 年原版** → 部分高频微观结构内容已过时（如 Reg NMS 后续修订、暗池监管演进）
- **数学要求高** → 需要测度论 / 随机分析 / SDE 基础（附录 A/B 仅做提醒，不充分）
- **不讨论工程纪律** → IS/OOS / 滑点 / 涨跌停 / 数据泄露等关键工程问题需自行配套
- **A 股本土化未涉及** → 涨跌停 / T+1 / 散户主导 / 监管特点未讨论
- **代码完全缺失** → 与 OpenAlphas / QuantsPlaybook 形成对比
