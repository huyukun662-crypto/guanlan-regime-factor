# 另类 ETF 交易策略：日内动量

> 来源：西部证券《指数化配置系列研究（1）》2024-08-09 | QuantsPlaybook 复现：[[hugo2046]]

## 一句话定位

借鉴论文《Beat the Market: An Effective Intraday Momentum Strategy for S&P500 ETF (SPY)》（Zarattini, Aziz, Barbon），构建基于**噪声区域突破**的 ETF 日内动量策略，**不持仓过夜**，针对上证 50 / 沪深 300 / 中证 500 / 中证 1000 ETF 做分钟级交易。

## 核心机制

### 1. 噪声区域 (Noise Area) 定义

> 当价格在"过去 14 天相同时间点的平均位移"内波动 → 视为供需平衡 → 不交易

**位移**（相对开盘价的收益率绝对值）：
```
move_{t-i, 9:30→hh:mm} = | close_{t-i, hh:mm} / open_{t-i, 9:30} - 1 |,    i = 1..14
```

**14 天均值**：
```
σ_{t, hh:mm} = (1/14) Σ move_{t-i, 9:30→hh:mm}
```

**噪声区域上下边界**（结合昨收盘缺口）：
```
UpperBound_{t, hh:mm} = max(open_{t,9:30}, close_{t-1,15:00}) × (1 + σ_{t, hh:mm})
LowerBound_{t, hh:mm} = min(open_{t,9:30}, close_{t-1,15:00}) × (1 - σ_{t, hh:mm})
```

### 2. 信号规则

- **价格突破上边界** → 多头需求 → 做多（次分钟开盘开仓）
- **价格突破下边界** → 抛售压力 → 做空（次分钟开盘开仓）
- **价格在区域内** → 无信号

### 3. 平仓规则（不持仓过夜）

- 多头持仓 + 价格跌破下边界 → 次分钟开盘平仓
- 空头持仓 + 价格突破上边界 → 次分钟开盘平仓
- 当日收盘时强制平仓

### 4. 交易时点

为避免边界震荡导致频繁信号，**仅在 10:29 / 11:29 / 13:59 三个时点**判断是否开仓。但平仓信号是即时触发（防止 stop-loss 滞后）。

## 改进版本（策略升级）

### 改进 1：VWAP 止损线
原始策略只用对向边界做止损 → 改为 **max(同向边界, VWAP)** 作为止损线（多头）/ **min(同向边界, VWAP)**（空头）。

VWAP（成交量加权平均价格）是另一个识别日内供需平衡变化的有效指标。

## 经济直觉

- 日内动量源于**粘性供需失衡**（sticky demand/supply imbalances）
- 价格对新信息的**反应不足**（论文经典假设）
- 持续突破"过去同时点平均波动" → 表明力量失衡持续 → 趋势可延续到日内

## 适用 / 不适用

✅ ETF（流动性好的宽基）+ 分钟数据
✅ 不持仓过夜 → 隔夜风险为零
✅ 日内动量是个被广泛验证的现象（中美都成立）
⚠ 需要分钟级数据（V25 项目目前是日频，需扩展数据源）
⚠ 14 天滚动窗口对参数敏感
⚠ 万一手续费假设乐观，实盘可能更高
