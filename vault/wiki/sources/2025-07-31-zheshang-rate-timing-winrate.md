# 胜率视角下的利率交易择时策略

> 来源：浙商证券债券市场专题研究 | [[覃汉]] / [[章恒豪]] | 2025-07-31 | 15 页 | 利率量化择时系列**二**

## 一句话定位

**前篇（赔率视角，左侧）= XGBoost 集成基本面分类模型 → 本篇（胜率视角，右侧）= 5 个子策略 + "任意 2 个信号触发"集成方法**。落地标的：T（10 年国债期货）+ TL（30 年国债期货）。

## 核心观点

### 一、赔率 vs 胜率的方法论二分

| 维度 | 赔率视角（左侧 / 系列一） | 胜率视角（右侧 / 系列二） |
|---|---|---|
| 数据 | 宏观基本面 + 事件 | 量价 + 技术指标 |
| 介入时点 | 趋势尚未确立（预判） | 趋势确立初期或中段（确认） |
| 优势 | 抄底逃顶、赔率高 | 胜率高、信号连续、执行清晰 |
| 劣势 | 预判错误损失重 | 滞后性强，易踏空 |
| 模型 | XGBoost 三分类 | 5 子策略 + 多信号融合 |
| 已发表 | 《赔率视角下的 30 年国债期货择时策略》 | 本篇 |

### 二、传统技术指标的边界（K 线 / MACD / KDJ）

- **K 线**：弱结构识别，依赖趋势背景 + 量能；利率期货日内波动收敛 → 仅在突发事件后剧烈波动日的次日有信号意义
- **MACD**：滞后 → 入场点常处趋势中段甚至末端；T 上盈亏比偏弱，TL 上假突破多
- **KDJ**：节奏敏感 → 震荡市误报多；TL 上 J 值常极端跳动

⇒ **共同问题**：滞后、误报、参数移植性差 → 难以独立构建可执行系统

### 三、5 个子策略（"趋势识别 - 节奏控制 - 信号整合"三层结构）

| 子策略 | 功能定位 | 核心机制 | 核心参数 |
|---|---|---|---|
| **[[低延时趋势]]** | 趋势识别 | 拉普拉斯/傅里叶变换 → 频域滤波去高频 → 复频域中识别趋势斜率 | α 权重、回溯窗口、斜率阈值 |
| **[[阶矩择时]]** | 波动结构 | EMA/HMA/FRAMA 平滑日收益率 3 阶矩；切线法选最优 α | α 权重、回溯窗口、动量阶数 |
| **[[单/双均线策略]]** | 趋势/通道确认 | 单线 = 价格偏离度；双线 = 金叉死叉 + 通道过滤 | EMA10 vs EMA30，布林带宽 |
| **[[通道过滤]]** | 波动结构 | 布林带突破 + 滚动 K 线有效突破识别 | 通道窗口、标准差倍数 |
| **[[九转序列]]** | 结构衰竭 | TD Sequential：收盘价低于前 4 日，连续 N 天即反转信号 | N 值、低价计数、量能确认 |

### 四、多信号策略 = "任意 2 个信号触发即入场"

- 类似机器学习的 **集成模型（ensemble）**
- 持仓规则：交叉检验信号触发法
- 优势：降低交易频率（节约成本）+ 子策略优势互补 + 在多信号开展交叉检验

### 五、回测结果（无滑点 / 杠杆 / 保证金）

| 指标 | T (2015/3/20-2025/7/29) | TL (2023/4/21-2025/7/29) |
|---|---|---|
| 年化收益 | 8.63% | **23.22%** |
| 累计收益 | 128.72% | 57.59% |
| 夏普比 | 3.04 | **3.78** |
| 最大回撤 | 2.62% | 2.28% |
| 持仓总天数 | 1736 | 430 |
| 交易次数 | 108 | 26 |
| 平均持仓天数 | 16 | 16 |
| **胜率（次）** | 70.37% | **80.77%** |
| 胜率（天） | 59.50% | 64.42% |
| 平均盈亏比（天） | 1.59 | 1.87 |

### 六、典型规避案例（TL，2025）
- 2025 年 2 月 + 7 月利率快速上行期 → 多信号策略**主动避开**
- 趋势识别 + 动量过滤 → 及时避免错误入场
- 极端行情下显示防御能力

### 七、后续优化方向（作者自述 3 项）
1. **信号权重机制** — 趋势类（低延时 / 双均线）vs 结构类（九转 / 布林带）分层加权
2. **数据维度扩展** — 引入新闻文本 / 政策文本 + LLM（GPT 微调 / DeepSeek）情感打分
3. **集成层升级** — 从规则法则 → Stacking / Voting Ensemble 弱监督学习

## 风险提示与待验证

- **回测假设**：无滑点、无杠杆、无保证金 → 实盘需扣减后再判断
- **TL 样本期短**（2.3 年） → 23% 年化的置信区间宽
- **T 样本期长**（10 年） → 8.63% 年化更可信
- 子策略参数 α、回溯窗口、阈值在每个回测日重新优化 → **可能存在过拟合风险**（90 日滚动优化是否够样本外稳定？）
- 后续优化中提到的 LLM 情感打分仍属"愿景"，未在本篇验证

## V25 / 复现 Hook

- 利率期货是 V25 项目目前未覆盖的标的（V25 主要做股票 / ETF）
- 但 5 个子策略中的 **低延时趋势 + 阶矩择时 + 九转** 都可迁移到 ETF 池
- 待办：在 V25 项目下创建 `opt/zheshang_winrate_replicate.py`
  - 优先实现 **九转序列**（最简单，TD Sequential 标准实现）
  - 然后 **低延时趋势**（拉普拉斯 / 傅里叶变换需要 scipy.signal）
  - 阶矩择时较复杂，可后置
  - 多信号融合的"任意 2 个信号触发"逻辑用 numpy 即可
