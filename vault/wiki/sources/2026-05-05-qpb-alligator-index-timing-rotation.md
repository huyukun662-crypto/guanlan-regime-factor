# 基于鳄鱼线的指数择时及轮动策略

> 来源：招商证券《金融工程：基于鳄鱼线的指数择时及轮动策略》2024-05-07 | QuantsPlaybook 复现：[[hugo2046]]

## 一句话定位

把比尔·威廉斯 1995 年《证券混沌操作法》中的**鳄鱼线（the alligator）+ AO 动量震荡 + 分形（Fractal）**经典技术分析体系迁移到 A 股**指数择时**。沿用原书参数（避免过拟合），扩展到 6 个核心宽基指数。

## 核心机制

### 1. 鳄鱼线（三条均线）

| 线 | 比喻 | 计算 |
|---|---|---|
| **下颚（蓝线）** | 鳄鱼下颌 | 13 根价格柱的 SMA + 向右平移 8 根 |
| **牙齿（红线）** | 鳄鱼牙齿 | 8 根价格柱的 SMA + 向右平移 5 根 |
| **唇吻（绿线）** | 鳄鱼嘴唇 | 5 根价格柱的 SMA + 向右平移 3 根 |

参数 (13, 8, 5, 3) 全部是**斐波那契数列**。

### 2. 三种鳄鱼状态

| 状态 | 排列 | 操作 |
|---|---|---|
| **饥饿的鳄鱼**（多头） | 上唇 > 牙 > 下颚 | 满仓做多 |
| **吃饱的鳄鱼**（空头） | 上唇 < 牙 < 下颚 | 空仓 / 做空 |
| **沉睡的鳄鱼**（震荡） | 三线纠缠 | 维持上一日仓位（不操作） |

### 3. AO 动量震荡指标（Awesome Oscillator）

**威廉斯**：通过 AO 指标交易者可以清晰地观察市场的走势

**原书公式**（也是 ipynb 实现）：
```
AO = SMA(High - Low, 5) - SMA(High - Low, 34)
```
（注意 TradingView 是 `SMA(High + Low, 5) - SMA(High + Low, 34)`，与原书不同）

**用法**：AO 连续 3 个交易日上行 = 加仓信号；下行 = 减仓信号

### 4. 分形突破信号（Fractal）

- **上分形**：5 根连续 K 线，最高点左右各有 2 根较低高点
- **下分形**：对称定义
- **信号**：T 日收盘价突破最近上分形高点 → 看多；跌破最近下分形低点 → 看空

### 5. 北向资金信号（额外引入）

仓库还引入了 [[北向资金]] 作为额外维度（具体细节未在节选中展开）。

## 信号融合（关键不对称设计）

```
看多信号 = AND(Alligator=1, OR(AO=1, Fractal=1))    # 苛刻：需鳄鱼多头排列 + AO/分形之一
看空信号 = OR(Alligator=-1, AO=-1, Fractal=-1)      # 宽松：任一信号触发
```

**不对称的原因**：A 股博弈较多 → 假突破多 → 看多需更谨慎；下跌迅速 → 看空需更及时。

## 实测结论

- 鳄鱼线**未能完全复现研报回测指标**（即使不加手续费）— 作者明确标注
- 加手续费（万 1.5）后收益下滑显著
- AO + 分形 + 鳄鱼线的组合提升了**胜率**和**盈亏比**

- **A 股不对称信号设计**（看多严格 / 看空宽松）是个有价值的工程细节
- 经典技术分析在 A 股有效性需通过严格回测验证（仓库作者诚实地标注了"未完全复现"）
- 与 [[多信号策略]] 的"任意 2 信号触发"思路一致 — 都是用多重信号互验
