# 行业指数顶部和底部信号（NH-NL%）

> 来源：华福证券《市场情绪指标专题（五）》2023-03-02 | QuantsPlaybook 复现：[[hugo2046]]

## 一句话定位

用**净新高占比 (NH-NL)%**（行业内创年度新高个股数 - 创年度新低个股数）/ 行业内上市超 1 年个股数 衡量行业指数的强弱与情绪。**对科技板块表现最好**（电子 / 通信 / 计算机 / 传媒）。

## 核心公式

```
(NH-NL)% = (行业内创 52 周新高数 - 创 52 周新低数) / 行业内上市超 1 年个股数
```

**两个细节**（与传统不同）：
1. 用收盘价 close 而非最高/最低价 high/low
2. 识别前期高点的窗口为 T_{1-52}（不含今日）而非 T_{0-51}

## 阈值（5 档情绪分类）

**标准阈值**（行业内成分股 ≥ 40 只）：

| (NH-NL)% | 情绪 | 含义 |
|---|---|---|
| ≥ 30% | **贪婪** | 顶部信号 |
| 20%-30% | 乐观 | |
| -20% ~ 20% | 正常区间 | |
| -30% ~ -20% | 悲观 | |
| ≤ -30% | **恐惧** | 底部信号 |

**小行业阈值**（< 40 只成分股，防止跳跃）：阈值放宽到 ±30% / ±40%

## 反转策略（简单实现）

- **贪婪信号** → 第一天回到阈值内 → **次日开盘做空**
  - 止损：前一周（5 日）区间最高点
  - 持有 30 天后收盘平仓；入场 1 周后未止损 → 止损位移到成本价
- **恐惧信号** → 第一天回到阈值内 → **次日开盘做多**
  - 止损：前一周区间最低点
  - 同样逻辑

## 经济直觉（基于行为金融学锚定效应）

1. **锚定效应**：投资者记得自己的成本价 → 创新高个股持有者均浮盈，抛压小；创新低个股持有者均浮亏，反弹即抛
2. **新高/新低是个股走强/走弱最直接的刻画**
3. **新高新低差代表行业整体强弱**
4. **归一化（指标值在 [-1, 1]）去除行业规模影响**（机械 ~500 个股 vs 煤炭 ~36 个股）

## 适用与不适用

✅ **科技板块**最好（电子 / 通信 / 计算机 / 传媒）— 报告复盘验证
✅ 周期板块表现也较好
⚠ 行业每半年调整成分股 → 复现时用最近截面成分股可能引发前序行情失真
⚠ 反转策略：资金占用时间较少 → 适合作为**指数增强型策略**（不适合作为主策略）
