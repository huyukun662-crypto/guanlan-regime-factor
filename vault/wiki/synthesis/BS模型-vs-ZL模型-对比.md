# BS 模型 vs ZL 模型 对比分析

> 对比 2 个对象 | 综合自 1 篇素材 + 6 个相关实体页 | 生成日期：2026-05-05

## 对比对象

- [[BS模型]] — Black-Scholes 静态欧式期权解析定价（"进攻型定价锚"）
- [[ZL模型]] — Zheng-Lin 蒙特卡洛 + 条款博弈最优停时（"防守型绝对价值锚"）

## 对比表

### 一、理论结构

| 维度 | [[BS模型]] | [[ZL模型]] |
|---|---|---|
| 定价框架 | 无风险套利均衡（连续时间） | 信用底仓 + 多重内嵌奇异期权（结构化产品） |
| 数学结构 | 闭式解析解（Black-Scholes 公式） | 数值解（[[蒙特卡洛模拟]] + 最优停时 / 动态规划） |
| 标的资产建模 | 风险中性下的几何布朗运动 | 风险中性下的几何布朗运动 + 路径模拟 |
| 期权类型假设 | 标准欧式看涨期权 | 美式期权 + 多重路径依赖条款 |
| 公式核心 | $V_{CB} = V_{bond} + V_{option}$（线性叠加） | 六步流程：模拟 → 触发 → 回售 → 强赎 → 循环 → 平均 |
| 信用利差处理 | 含信用风险的贴现率 $r_d$ | 严格施加信用利差贴现 + 自动剔除低评级标的非理性溢价 |

### 二、条款覆盖

| 条款 | [[BS模型]] | [[ZL模型]] |
|---|---|---|
| [[强赎条款]] | **缺失刻画**（核心局限） | 显式处理（路径终止 + 价值锁定 $\text{Max}(S_t \times FV/X, B_c)$） |
| [[下修条款]] | 缺失刻画（被强赎缺失抵消） | 显式处理（用 BS 公式反算最优下修幅度，存在轻微低估） |
| [[回售条款]] | 缺失刻画 | 显式处理（触发回售 → 假定发行人主动下修） |
| 路径依赖 | 无（静态截面） | 有（动态路径追踪） |

|---|---|---|

### 四、定价偏差规律

| 维度 | [[BS模型]] | [[ZL模型]] |
|---|---|---|
| 与 [[在值程度]] 关系 | 偏离随 Moneyness 提升单边放大（深实值高估明显） | 整体贴近但略低于市场，强股性区段触发强赎机制 |
| 与剩余期限关系 | 长久期（>1400 天）高估 / 中短端（200-1200 天）低估 / 极短端（30-50 天）收敛 | 全期限稳定低估 + 与市场趋势同步 + 临近到期收敛 |
| 偏差驱动因素 | 缺失天花板 + 常数波动率假设外推 | 信用利差贴现 + 严格条款边界 + 下修幅度估计偏差 |

| 维度 | [[BS模型]] | [[ZL模型]] |
|---|---|---|
| 计算成本 | **极低**（闭式解，可大规模实时定价） | 高（需 GPU 并行 + 大量路径模拟） |

## 关键差异

**一句话总结**：BS 模型用"忽略复杂条款"换来"计算高效 + 牛市进攻弹性"，ZL 模型用"算力换精度"换来"条款博弈刻画 + 熊市防守威力"——两者不是替代关系而是**宏观状态切换下的互补工具**。

**最关键的三个分化点**：

1. **强赎刻画**：BS 缺失 → 牛市系统性高估（伪超额收益）；ZL 显式处理 → 估值收敛（真实底线）
2. **风险收益曲线**：BS 高收益 + 高回撤；ZL 中等收益 + 极低回撤 + 最高夏普
3. **状态依赖**：BS 极度依赖估值扩张红利（熊市完全失效）；ZL 在牛 / 熊都能稳定贡献，下行周期空头威力尤为突出
