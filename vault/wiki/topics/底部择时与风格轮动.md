# 底部择时与风格轮动

> 在 A 股牛熊周期底部区域，通过量化指标识别"底部确认"并触发"价值 → 成长"风格切换的研究主题。

## 核心问题

1. **怎么识别底部** — 单指标易误判，需多因子合取
2. **底部内部如何分阶段** — 左侧 / 右侧初期 / 切换期，每阶段最优配置不同
3. **切换信号怎么定义** — 用什么变量、什么阈值、什么置信度
4. **历史规律是否稳定** — 样本量、置信区间、样本外验证

## 研究内容地图

### 底部识别因子

- [[底部双因子识别]]：MDD ≥ 20% & PE_percentile ≤ 10%
- [[上证综指3年滚动MDD]]：价格条件
- [[上证综指PE分位]]：估值条件

### 风格切换因子

- [[SRI]]：成长 / 价值收盘价比值（核心锚点）
- [[SRI_balance]]：(SRI_max + SRI_min) / 2 ≈ 0.83（切换阈值）

### 阶段框架

- [[风格轮动三阶段模型]]：左侧 → 右侧初期 → 切换期

### 风格分类

- [[价值风格]] vs [[成长风格]]
- [[国证成长指数]] vs [[国证价值指数]]

### 评估口径

## 待跟进研究方向

1. **样本外验证**：本 OpenAlphas 模型用 2005-2022 数据，需 2023-2026 数据外推确认
2. **跨数据源复现**：用 efinance 替换 akshare，验证 SRI / SRI_balance / 双因子触发是否一致
3. **与可转债事件驱动的耦合**：MDD + PE 双因子触发的"底部" 是否与可转债下修概率提升相关
4. **板块绑定方案**：把 V25 ETF 池按价值/成长打标，配合 SRI 信号做加权倾斜
5. **更细颗粒度的底部识别**：MDD 与 PE_percentile 的"AND" 是否过严？是否需要"OR + 软阈值"？
