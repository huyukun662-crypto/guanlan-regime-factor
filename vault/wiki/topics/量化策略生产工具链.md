# 量化策略生产工具链

> 把 alpha 挖掘从"研究员手工脚本"升级成"工业流水线"的工具与机制。

## 核心问题

量化策略生产中最稀缺的不是因子点子，而是**把点子大规模、自动化、可重现地跑出来**的工程能力：
- 并发回测槽位
- 大规模任务切片 / 断点续传
- 自动筛选去冗余
- 鲁棒性压力测试
- 提交达标的硬约束

## 平台 / 工具

| 平台 | 类型 | 备注 |
|------|------|------|
| [[WorldQuant BRAIN平台]] | 商业 alpha 平台 | 用 Python API + 表达式 DSL |
| [[OpenAlphas]] | 开源 alpha 库 | 思路对照、非平台 |
| [[Qlib]] | 微软开源量化框架 | 本地完整流水线 |

## 关键对比

| 维度 | BRAIN 平台 | 自有 V25 ETF 轮动 |
|------|-----------|------------------|
| 回测吞吐 | 80 并发（顾问） | 取决于本地机器 |
| 因子库 | 自构表达式或 Python | 手写因子 |
| 鲁棒性测试 | rank/sign/binary 三件套 | 自有 IS/OOS 三段式 |
| 提交达标 | Sharp 58 + fitness 1（顾问硬约束） | 无外部约束 |
