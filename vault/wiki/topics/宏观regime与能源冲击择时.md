# 宏观regime与能源冲击择时

> 自上而下的**宏观 regime 识别**与能源供给冲击下的资产价格节奏研究——为自有自下而上的价量择时提供 regime 门控 overlay。

## 主题定位

本库主体是自下而上的因子 / 价量择时（[[技术指标择时]]、[[底部择时与风格轮动]] 等）。本主题收集**互补的自上而下视角**：用宏观 regime（滞胀 / 衰退 / 复苏）划分市场状态，作为权益仓位与防御切换的门控。当前以能源冲击 / 滞胀场景为切入点。

> 注意：本主题素材多为卖方宏观叙事（B 档，无回测），价值在**框架与可量化 regime 指标**，纳入自有体系前一律 IS/OOS 独立验证。**仅供研究参考。**

- [[滞胀交易三阶段框架]] — 能源冲击下资产价格三段式 regime 模板（仅胀无滞 → 滞胀并至 → 科技接棒）

## 可量化 regime 指标

| 指标 | 口径 | regime 含义 |
|---|---|---|
| [[金铜比]] | gold/copper 滚动 z-score | 上行加速 → 衰退交易（增长担忧 > 工业需求） |
| [[期限利差倒挂]] | UST 10Y-2Y | 倒挂深化 → 衰退确认 |
| [[OVX原油波动率指数]] | OVX 水平 + z-score（类 VIX） | 高位 → 油价风险溢价；快速回落 → 风险偏好修复 |
| [[跨能源比价均值回归]] | oil/coal、oil/gas 比价 z-score | 偏离 >1σ → 中长期均值回归 RV 信号 |
| [[原油库存可用天数]] | days-of-supply（~20 天警戒） | 逼近警戒 → 供给脆弱、油价上行尾部风险 |
| 通胀盈亏平衡率 | TIPS breakeven 1Y/10Y | 走高 → 通胀失锚风险 |
