# 价值投资量化

> 把价值投资的"竞争壁垒+优质标的"逻辑落地为量化框架

## 主题定位

把巴菲特 / 芒格的价值投资经典理论（企业价值 = 资产价值 + 盈利能力价值 + 成长价值）落地为**可量化、可复现的两层筛选流水线**。区别于：
- 传统价投：依赖深度行业研究、估值主观、持仓周期长
- 传统因子投资：大数定律 + 广撒网 → 高换手 / 风格切换易失效
- ML 选股：黑箱预测，难以与基本面研究互通

价值量化的核心特征：**低换手、高胜率、高赔率、可与主观研究对话**。

## 关键素材

- [[2026-05-05-openalphas-value-quant-revolution]]：先定行业壁垒 → 再选优质标的，年化 40%+ 宣称（AMBIGUOUS）

## 核心概念

### 第一层：行业分类
- [[行业格局稳态性指标]] S = 1 - percentile(median(D_{T-11..T}))
- [[行业盈利可持续性指标]] P = percentile(median(ROE_{T-11..T}))
- 阈值：S > 0.7 且 P > 0.7 → **高壁垒稳态行业**；其余 → **充分竞争行业**

### 第二层：标的筛选
- 高壁垒行业：绝对龙头 / [[寡头共赢标的]]
- 充分竞争行业：[[高效运营组合]]（成本控制 + 资产周转效率）

### 数据源
- [[akshare]] + 中信三级行业分类

## DCF 黑箱问题的解决

价值投资传统 DCF 估值的缺陷：用财务报表外推却忽略了**决定终局价值的核心变量**——企业的长期竞争壁垒。本主题的解决方案：
1. **行业层面用稳态性 + 盈利可持续性**两个指标共同识别"长期能挣钱的赛道"
2. **标的层面用财务质量 + 现金流质量 + 运营效率**多维筛选
3. 完全绕开 DCF 的主观折现率假设

## 优化方向

1. PB-ROE 估值匹配过滤（规避高估值标的）
2. 引入专利数据 / 品牌力 / 牌照资源等另类数据
3. 仓位管理（按宏观周期 / 波动率调整）
4. 流动性指标 → 适配大资金
