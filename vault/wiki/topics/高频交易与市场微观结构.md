# 高频交易与市场微观结构

> 围绕 [[限价订单簿LOB]] 的高频数据分析、最优执行、做市策略的理论与实践集合

知识库的**理论根基主题** — 之前的素材都是日频中频策略（OpenAlphas / 浙商 / 广发 / QuantsPlaybook），从 [[2020-quantitative-trading-textbook]] 教科书入库后，本主题**从微秒级到毫秒级到秒级**的市场微观结构维度被系统覆盖。

- **数学要求高**（HJB 方程 / Hawkes 过程 / 排队论）
- **填补 V25 项目的理论盲区**（V25 当前是日频，本主题指向未来扩展方向）

## 三大核心子领域

### 1. 限价订单簿（LOB）建模 — 第 5 章
- [[限价订单簿LOB]]（核心）
- 三类建模方法：
  - **多元点过程**（Hawkes 过程）— 捕捉订单聚簇
  - **机器学习**（深度学习预测短期方向）
  - **排队论**（M/M/1 + 扩散极限）
- **关键观测信号**：交易量不平衡 / 价差 / 订单深度

### 2. 最优执行 — 第 6 章
- [[最优执行]]（核心）— 大订单拆分
- 经典模型：
  - **Bertsimas-Lo 1998**（开创）
  - **Almgren-Chriss 2001**（事实标准）
  - **Obizhaeva-Wang 2013**（LOB 数据驱动）
- 实操算法：**TWAP / VWAP / POV / Implementation Shortfall**

### 3. 做市与流动性提供 — 第 7 章
- [[做市策略]]（核心）
- 经典模型：
  - **Ho-Stoll 1981**（库存风险驱动报价）
  - **Avellaneda-Stoikov 2008**（Mean-variance + 时间动态）
- 高级机制：
  - 脉冲控制（限价 + 市价混合）
  - SOR（Smart Order Routing）
  - 暗池（删失探索 / 贪心配置）
