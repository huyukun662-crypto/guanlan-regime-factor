# 利率期货量化择时

> 围绕 T（10 年）/ TL（30 年）国债期货的量化择时方法学体系

## 主题定位

[[浙商证券]] 债券研究团队的"利率量化择时系列"用**两条对照路径**覆盖利率期货择时问题：
- **系列一（赔率视角）**：基本面 / XGBoost 集成 → 左侧交易
- **系列二（胜率视角）**：技术指标 / 5 子策略 + 多信号融合 → 右侧交易

两套方法**在哲学上对立、在实战中互补**：赔率视角负责"做不做"，胜率视角负责"何时做"。

## 重要发现 / 启示

### 1. 胜率与赔率的方法论二分（最重要）
利率市场中两类策略各自适用不同环境：
- 基本面主导阶段 → 赔率视角占优（XGBoost 集成）
- 波动率抬升、趋势强化阶段 → 胜率视角占优（多信号融合）

### 2. 集成模型在择时中同样有效
本研究的"任意 2 信号触发"是**最简形式的 ensemble**，但配合精心设计的子策略已能跑出 70-80% 胜率 + 3.0+ 夏普。

### 3. 频域滤波的价值
传统时域均线本质上是 IIR 低通滤波器，但**滞后明显**。引入频域分析（拉普拉斯 / 傅里叶）能显著降低滞后。

### 4. 高阶矩信息被忽视
市场结构和行情节奏并不稳定，高阶矩（加速度及以上）具备前瞻性，但传统模型很少用。

### 5. TD Sequential 在 A 股 / 利率市场仍然有效
"九转底" / "九转顶" 是经典工具，在新兴市场（A 股）和长久期产品（TL）中**触发频率适中、信号清晰**。

## 待解决的问题

1. **赔率视角系列一报告未在知识库** — 缺失左侧策略的详细机制（XGBoost 三分类的特征工程 / 阈值设定）
2. **多信号策略的信号权重升级** — 作者提到的"分层加权"未给具体方案
3. **LLM 情感打分的接入** — 仍是愿景阶段
4. **TL 样本期短** — 23% 年化的稳定性需更长样本验证
5. **回测假设过宽** — 无滑点 / 杠杆 / 保证金，实盘需扣减后再判断

## V25 项目集成路径

1. **快速胜利**：实现 [[九转序列]] 标准 TD Sequential（10 行代码可完成）
2. **中等成本**：[[低延时趋势]] 用 scipy.signal 实现频域滤波
3. **进阶**：[[阶矩择时]] 用 scipy.stats.moment + 切线法寻最优 α
4. **集成**：用 numpy `(signals.sum(axis=1) >= 2)` 实现"任意 2 信号触发"
5. **回测对比**：在 V25 现有 ETF 池（不止国债）上测试是否能复现胜率 70%+
6. **补全方向性**：等待用户提供"赔率视角"系列一报告 → 实现完整左 + 右组合
