# 技术指标择时

> 用价格、成交量等技术指标判断"何时入场 / 何时离场"的择时策略

## 主题定位

技术指标择时关注**"什么时候交易"**而非**"交易什么"**。本主题汇总以 [[RSRS]] 为核心的现代量化择时方法 — 区别于传统的纯主观技术分析（裸 K / 黄金分割等），这些方法都有：
- 明确的数学公式定义
- 可回测验证的胜率 / 夏普
- 可工程化的过滤增强机制

## 关键素材

- [[2026-05-05-openalphas-rsrs-market-thermometer]]：A 档；RSRS 三层标准化 + MA20 / 成交量过滤，胜率 68%+ / 夏普 1.8+ 宣称（AMBIGUOUS）
- [[2026-05-19-jq-ama-kaufman-explainer]]：A 档；KAMA 自适应均线 + 效率系数 + 标准差过滤器，聚宽 TA-Lib 直接可调；缺业绩验证与 σ 系数

## 核心概念

### RSRS 体系
- [[RSRS]] — 阻力支撑相对强度（基础斜率 β）
- [[RSRS标准分]] → [[RSRS修正标准分]] → [[RSRS右偏标准分]]（三层标准化）

### 自适应均线体系（KAMA / Kaufman）
- [[考夫曼自适应均线]] — KAMA，用 ER 动态调节平滑速度（趋势快 / 震荡慢）
- [[效率系数]]（ER）— 净位移 / 路径长度 ∈ [0,1]，趋势强度归一化分类器
- [[标准差过滤器]] — 配合 KAMA 的"变速箱"，去毛刺去假突破
- [[AMA双均线策略]] — 简化版双 KAMA 金叉死叉模板

### 增强机制
- [[价格趋势过滤]]（MA20 趋势判断）— MDD 50%+ → 20%
- [[成交量相关性过滤]]（10 日成交量与 RSRS 修正标准分相关性）
- [[标准差过滤器]] — 信号阶段去毛刺的通用范式

## 传统技术分析的痛点

1. **突破信号滞后**：等确认突破时，行情已走完一半
2. **震荡市反复打脸**：假突破让你不断止损
3. **大熊市支撑位逐级失效**：抄底抄在半山腰

## RSRS 解决思路

**从"固定点位"转向"相对强度"**：
- 不再关注"是否突破阻力位"
- 关注"支撑/阻力的相对强度变化"
- 通过 OLS 回归 + 三层标准化 + 拟合可靠性加权得到领先信号

## 适用与局限

✅ **指数级别择时**（沪深300、上证综指）
✅ 中级别（周 / 月）趋势翻转判断
⚠ 不适合个股（高低价波动剧烈失真）
⚠ 不适合短期（日内 / 高频）
⚠ 极端震荡市仍可能亏损
⚠ 参数对结果有一定影响，需在不同市场上参数优化

## 优化方向

1. 多周期 RSRS 共振（日 / 周 / 月）
2. 跨指数 RSRS 信号合成（同时看大小盘）
3. 加入 VIX / 沪深 300 隐波过滤极端恐慌
4. 与 [[底部双因子识别]] 叠加，构建多级别择时框架
