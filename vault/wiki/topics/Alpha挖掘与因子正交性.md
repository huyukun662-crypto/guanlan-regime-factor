# Alpha 挖掘与因子正交性

> 验证新因子是否提供与传统风格因子不重叠的纯粹超额收益的方法学体系。

## 核心观点

1. **正交性是新因子的入场券**：与传统因子相关系数低（典型阈值 |ρ| < 0.25 ~ 0.3）说明因子捕获的信息源不同，能避免多重共线性的冗余暴露。
2. **正交性 ≠ Alpha 纯度**：相关系数低只能说明线性相关性弱，需进一步用 [[双层分组分析]] / [[风格中性约束]] 在严苛切片下检验目标因子是否仍输出超额。
3. **Alpha 来源的可解释性**：通过证伪传统认知（如"BS 高收益是高波 / 强动量替代品"），明确 Alpha 真正来源（如 BS / ZL 因子的纯粹期权价值回归）。

## 方法论流程

1. **第一步：相关性检验** — 与传统风格因子做截面相关系数热力图
2. **第二步：单因子绩效** — 在控制交易成本下的累计超额、夏普、回撤
3. **第三步：多因子合成** — 验证因子在等权 / 加权合成中是否带来阶跃式提升
4. **第四步：风格中性验证** — 通过双层分组排除隐性风格暴露替代品的可能
5. **第五步：解释 Alpha 来源** — 给出经济学 / 数理逻辑解释
