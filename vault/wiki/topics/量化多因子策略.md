# 量化多因子策略

> 通过构建相互正交的因子组合并对其等权 / 加权合成，提升信息比率（IR）的量化投资范式。

## 核心观点

1. **因子价值的核心**：不仅在于自身单维度预测能力，更在于能否提供有别于传统风格因子的**增量超额收益**（即正交性）。
2. **多因子合成的工程实践**：
   - 引入双边交易成本（典型 0.06%）剔除换手率过高导致的伪收益
   - 等权或基于历史夏普加权
   - 月度滚动调仓
3. **因子纯度提纯**：必须通过 [[双层分组分析]] / [[风格中性约束]] 检验，排除"伪 Alpha"（隐性风格暴露的替代品）。

## 关键概念

- [[错误定价因子]] — 由 [[BS模型]] / [[ZL模型]] 衍生的另类 Alpha 因子
- [[SRI]] — 风格相对强度，新增指数级风格择时因子
- [[底部双因子识别]] — MDD + PE 分位的合取，作为风格因子的触发条件
- [[双层分组分析]] — Double-Sort，因子纯度检验方法
- [[风格中性约束]] — 排除隐性风格暴露的检验范式
- 五大传统风格因子：流动性、波动率、量价相关性、估值、动量

## 双因子定位与组合角色

- **进攻型**：BS 偏差因子（推升组合年化收益上限）
- **防守型**：ZL 偏差因子（优化组合夏普 + 平抑尾部回撤）
- **理想合成**：高弹性因子（量价、波动率）负责向上进攻 + ZL 因子负责封死下行空间

## 未解决的问题

- 当前研究仅在等权合成下展示绩效，**基于历史夏普加权或风险预算的最优合成**未做对比
- 不同宏观状态（牛 / 熊 / 震荡）下的**因子动态权重切换**机制是开放方向
