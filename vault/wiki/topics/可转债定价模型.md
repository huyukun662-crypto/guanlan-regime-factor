# 可转债定价模型

> 在中国可转债市场"高估高价、超买生态"下，构建系统性剥离情绪溢价的绝对定价框架，将错误定价转化为可交易 Alpha。

## 核心观点

1. **传统相对估值失效**：在转债估值中枢全面抬升、高价券占比结构性提升的环境下，平价溢价率等相对估值指标极易被市场情绪、交易拥挤度及微观供需结构所扭曲。
2. **绝对定价框架的核心价值**：
   - **降维重构**：将复杂条款与路径依赖压缩为可量化测度的理论基准
   - **风险刻画**：动态刻画不同 [[在值程度]] 与期限结构下的底层风险暴露
   - **交易转化**：将"错误定价"提纯为确定性、可交易的 Alpha 信号
3. **两类主流框架的分化定位**：
   - [[BS模型]] = 进攻型定价锚（无风险套利、欧式期权解析解、对正股趋势 + 波动率敏感）
   - [[ZL模型]] = 防守型绝对价值锚（[[蒙特卡洛模拟]] + 条款博弈最优停时）
4. **状态切换配置范式**：
   - 趋势上行 + 估值扩张期 → 重仓 BS 多头，最大化捕获弹性与情绪溢价
   - 宏观流动性收紧 + 信用风险频发的下行震荡市 → 全面切换至 ZL，利用其精准底线定价与敏锐做空识别能力

## 关键概念

- [[BS模型]] — 静态欧式期权解析解，进攻型
- [[ZL模型]] — 蒙特卡洛 + 条款博弈最优停时，防守型
- [[可转债]] — 标的，兼具债性 + 股性
- [[蒙特卡洛模拟]] — ZL 模型实现机制
- [[强赎条款]] / [[下修条款]] / [[回售条款]] — 三大核心内嵌期权
- [[在值程度]] — 定价偏差的关键解释变量
- [[错误定价因子]] — 模型理论价格与市场价格的相对偏差，按月滚动调仓
- [[事件驱动策略]] — 后续优化方向

## 海内外定价研究脉络

- **海外起源**：以公司价值为基础变量的结构化模型 → 以股价为锚的简约式模型 → 融入信用风险（TF 模型）与利率风险精细化刻画
- **中国本土特色**：发行人促转股意愿极强，下修条款独具特色，路径依赖 + 非线性强赎约束
- **本土研究演进**：
  1. **数理机制深化**：保留 BS 风险暴露刻画 + 引入二叉树 / LSMC 处理美式期权最优停时和复杂条款
  2. **工程算力跃升**：测度变换降方差 + Tensor / GPU 并发 + MLP 神经网络融入蒙特卡洛
  3. **前沿方向**：从静态截面定价 → 动态概率调整的 [[事件驱动策略]]

## 未解决的问题

- **强约束下空头端逻辑被屏蔽**：在 AA- 及以上评级 + 高流动性筛选后，被判定"高估"的标的往往是被资金抱团的优质标的，做空在 A 股环境胜率与赔率极低 → 需要放宽至全市场才能验证空头威力
- **下修幅度估计偏差**：[[ZL模型]] 用 [[BS模型]] 公式反算最优下修幅度，存在系统性低估；如何用 Logistic / 神经网络更精准预判下修概率是开放问题
- **BS 模型在熊市完全失效**：2022Q1–2024Q3 下行周期 BS 多头年化 -5.35%、夏普 -0.45 → 进攻型锚在风险偏好收缩期的应对方案待补强（除"切换至 ZL"外）
