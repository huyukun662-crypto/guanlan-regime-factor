# 动态资产配置与组合优化

> 把信号（factor scores / regime 概率 / anchor）转化为可执行权重的工程层。

## 核心问题

1. **协方差估计**：高维下样本协方差不稳定，怎么正则化？
2. **目标函数**：纯 MV 太敏感，加什么惩罚项？
3. **换手率控制**：input 微变如何不带来过度交易？
4. **稳定性 vs 敏感性**：模型对新信息响应多快才合适？

| 方法 | 解决什么 | 关键引用 |

| [[L1换手率惩罚]] | 稀疏化权重变化 | 同上 |

## 工程化要点（从 Yang 2026 总结）

1. **协方差用 Ledoit-Wolf 不用样本估计**（高维下稳）
2. **加 L1 turnover 项**（控交易成本 + 输入鲁棒性）
3. **anchor 不作 predictor 只作约束**（防过拟合）
4. **expanding window 月度 refit**（适应环境变化但避免过拟合）
5. **clipping bound [0.5, 1.5]**（防极端事件冲击）
