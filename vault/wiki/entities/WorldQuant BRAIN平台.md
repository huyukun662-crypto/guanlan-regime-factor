# WorldQuant BRAIN平台

## 定义

WorldQuant 公司开放的商业 alpha 因子挖掘与提交评估平台，用户/顾问提交表达式或 Python 脚本，平台跑回测、给出 Sharp/fitness 评分，达标的 alpha 进入 WorldQuant 实盘组合。

## 核心要点

- 双角色：用户（业余）+ 顾问（付费/合作）
- 核心 API：数据拉取 / 表达式回测 / 提交检查
- 评估指标：Sharp、fitness、turnover、margin、PnL 曲线
