# 有限注意力（Limited Attention）

> 投资者注意力是稀缺资源 —— 单次信息冲击不够大就不会触发关注，离散大幅信息被充分定价、连续小幅信息被系统性忽视

## 一句话定义

行为金融经典假设：**人的注意力存在下限阈值**。在金融市场中，这导致**信息以大幅离散方式到达 → 充分（甚至过度）反应；以小幅连续方式到达 → 反应不足**——这是 [[FIP效应]] 的核心解释机制。

## 在 FIP 论文中的具体表述

- 一只股票每天涨 0.2% → 没人会写分析报告
- 某天暴涨 8% → 分析师、媒体、社交网络蜂拥而至
- 结果：**离散信息被充分定价，连续信息被系统性忽视**

## 强化条件（[[Da-Gurun-Warachka 2014]] Table 3）

| 条件 | FIP 效应 |
|---|---|
| 机构持股越分散（无大股东盯盘） | **更强** |
| 媒体覆盖越少（没记者写报道） | **更强** |
| 分析师覆盖越少 | **更强** |

> 注意力越分散 → 连续信息被忽视的程度越深 → FIP 套利空间越大。

## 为什么这构成"宝藏"

> "不受关注的品种往往是动量策略的宝藏——不是因为它们'便宜'，而是因为它们的信息没有被充分定价。"

- 不是估值便宜（PE / PB 低）
- 而是**信息流的边际买盘没到位**
- 后续买盘是"补作业"，定价过程是"低估的持续修正"
- 因此动量更强、更持久、不反转

## A 股 ETF 的天然契合 (INFERRED)

A 股 ETF 多数品种（特别是行业 / 主题 / 商品 ETF）：

✅ 没有大股东盯盘（散户和量化基金主导）
✅ 几乎无媒体覆盖（"豆粕 ETF 涨 0.3%" 不会上新闻）
✅ 分析师覆盖稀少（不像个股有几十份研报）

→ **完美匹配 FIP 增强条件** → A 股 ETF 上 [[信息离散度ID]] 因子的潜在 alpha 可能比美股个股更强。

## 与其他行为金融概念的关系

### 与处置效应（Disposition Effect）

- 处置效应：投资者倾向于过早卖出盈利、过晚卖出亏损
- 有限注意力：投资者根本不关注小幅波动
- **关系**：FIP 论文证明 ID 的预测力**独立于**处置效应——分析师不存在处置效应，但同样在连续信息后预测误差更大（t = -2.19）

### 与 [[凸显理论]]（Bordalo-Gennaioli-Shleifer 2012）

- 凸显理论：极端收益扭曲投资者的决策权重
- 有限注意力：单次信息冲击不够大不被关注
- **共同点**：两者都强调"注意力 / 显著性"是定价偏差的源头
- **差异**：凸显理论关注**截面对比**（哪些标的的极端值最显眼），有限注意力关注**时序到达方式**（信息怎么来的）
- 实证：[[凸显性因子STR]] 在 A 股有效，[[FIP效应]] 应该也在 A 股有效（同一行为金融家族）

### 与 [[Hong-Lim-Stein 2000]]

- Hong-Lim-Stein 2000：坏消息扩散慢，特别是小盘 + 低分析师覆盖股票
- FIP 是其升级：不只是"扩散慢"，而是"**离散方式扩散得快、连续方式扩散得几乎消失**"

## 实操含义

### 选股 / 选 ETF

- 优先选**冷门、低关注度**标的（论文支持）
- 在这些标的上，慢涨的动量更可靠
- 避免"网红 ETF" / 媒体高频炒作的标的——FIP 效应已被消耗

### 信号设计

- 把"关注度代理"作为因子的**条件分组维度**：
  - 关注度低 + 慢涨 → 重仓
  - 关注度低 + 暴涨 → 警惕
  - 关注度高 + 慢涨 → 弱化（已被定价）
  - 关注度高 + 暴涨 → 主动减仓

### 关注度代理（A 股 ETF）

| 代理变量 | 数据源 | 含义 |
|---|---|---|
| 日均成交额 | 行情 | 流动性 ≈ 关注度 |
| 持仓基金数 | 公告 | 机构关注度 |
| 百度搜索指数 | 第三方 | 散户关注度 |
| 雪球讨论数 | 第三方 | 社交媒体关注度 |
| 媒体提及次数 | 新闻爬虫 | 媒体关注度 |
