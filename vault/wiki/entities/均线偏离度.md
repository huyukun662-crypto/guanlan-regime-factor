# 均线偏离度

> 又名：乖离率、Log Bias — [[刘晨明]]提出的用于追踪 A 股**主线行情**强弱的趋势指标

## 一句话定义

用收盘价对 20 日 EMA 的对数偏离衡量趋势强弱，阈值划分"过热 / 良性 / 转弱"区间，**对传统 MA 突破信号增加了一个数据驱动的过滤带**，主要用于追踪强势主线行情。

## 两个版本

### 1. 除法版（原创版）— 行业指数标定

来源：[[2026-05-05-gf-strategy-mainline-correction-vs-end]]（2025-09-14）

```
均线偏离度 = (对数收盘价 / 20日EMA) - 1
```

**阈值**：
- 入场：> 2% 不追、0.6%-1.8% 适中、< 0.6%/为负 转弱
- 离场：均线上方无忧、-0.6%~0% 坚守、< -0.6% 离场

**适用**：行业指数 / 中等绝对值的主线品种

### 2. 减法版（升级版）— ETF 友好

来源：[[2026-05-05-gf-strategy-residential-entry-thermometer]]（2025-09-21）

```
均线偏离度 = ln(close) - ln(ema20)         # 数值近似偏离百分比
```

**阈值（× 100% 后）**：
- 入场：> 15% 不追、5%-15% 适中、< 5%/为负 转弱
- 离场：均线上方无忧、-5%~0% 坚守、< -5% 离场

**适用**：ETF / 低绝对值标的（避免除法版分母接近 0 的报错 / 正负颠倒）

### 数学等价性

两版的核心都是计算 `ln(close) - ln(ema20)` 这个**对数差**，差别仅在：
- 除法版：`(ln(close) / ema20_of_log) - 1` ≈ ln(close)/ln(ema20) - 1（数值上接近 ln 差但带小项偏差）
- 减法版：直接 `ln(close) - ln(ema20)` 更稳定

**注意**：减法版若用 `Log` （base 10）而非 `Ln`（自然对数）会有巨大差别（数值差 1/ln(10) ≈ 0.434 倍）—— 来源博主 [[2026-05-05-lcm-bias-indicator-personal-notes]] 特别提醒。

## 通达信公式（个人复刻版）

```
EMA20:= EMA(LN(CLOSE), 20);
LOGBIAS: (LN(CLOSE) - EMA20) * 100;
过热线:    15, COLORRED,    DOTLINE;
失速线:    5,  COLORYELLOW, DOTLINE;
止损参考线: -5, COLORGREEN,  DOTLINE;
零轴:      0,  COLORGRAY,   DOTLINE;
```

## 双重过滤逻辑

1. **阈值过滤**（见上方阈值）
2. **趋势过滤**：最近 20 日中收盘价至少 10 日在均线上方，否则趋势已转弱

## 为什么用 EMA20 而非 MA20

- EMA 给予近期数据更高权重，趋势行情中**收盘价与 EMA 更贴合**
- 阿尔·布鲁克斯《高级趋势技术分析》也采用 EMA20 作为趋势支撑位辅助判断

## 个人博主的 5 条心得（[[2026-05-05-lcm-bias-indicator-personal-notes]]）

1. **不上 5% 不够强** — 主线必须突破 5%，否则缺乏龙头气质
2. **回抽两步走** — 触 5% → 回 0 → 再上 → 强势确立
3. **品种间择强辅助** — 同一时点对比不同主线乖离率
4. **止损后不轻易入场** — 跌穿 -5% 后，未能再上 5% 即真主跌浪
5. **逆练真经** — 价值流（银行/红利）反向当震荡区间用

## 适用 / 不适用

✅ **趋势性强的品种**：科技成长、AI 产业链、新能源车、半导体等
✅ 行业指数 + ETF（减法版）
✅ 主线行情的过热 / 调整 / 终结识别

⚠ 不适用：
- 中证白酒（趋势性弱，反弹中频骗线）
- 价值流（银行 / 红利） — 长期在 [-5%, 5%] 震荡，需反向使用
- 短期波动（日内 / 高频）

## 与其他择时指标的对比

| 指标 | 适用周期 | 信号特点 | 致命弱点 |
|---|---|---|---|
| **均线偏离度（本指标）** | 中级（周月） | 主线追踪 + 强弱判断（位置型） | 对趋势性弱的品种失效 |
| [[考夫曼自适应均线]]（KAMA） | 中级 | 均线自适应平滑速度（速度型） | 单标的、缺组合维度 |
| [[效率系数]]（ER） | 任意 | 趋势-震荡分类器 ∈ [0,1] | 对涨跌停跳空敏感 |
| [[RSRS]] | 中级（周月） | 阻力支撑相对强度 | 左侧抄底倾向 |
| [[底部双因子识别]] | 大级（年） | 价格 MDD + 估值 PE | 大底信号稀少（6 个事件） |

> **正交性**：均线偏离度衡量价格相对均线的**位置**（偏离百分比），KAMA 衡量均线本身的**速度**（自适应平滑），二者属于均线择时的两个正交维度，理论上可以叠加使用 <!-- confidence: INFERRED -->

## FIP 视角的 BIAS 解读（2026-05-08 追加）

[[FIP效应]] 给 BIAS 提供了独立的学术解释：

| BIAS 状态 | 涨法 | FIP 分组 | 6 月动量 | 操作 |
|---|---|---|---|---|
| 温和（5%-15%） | 慢涨 | 连续信息组（ID 低） | **8.86%** | 持有 / 加仓 |
| 飙升（> 15%-20%） | 暴涨 | 离散信息组（ID 高） | 2.91% | [[BIAS减仓]] |

→ BIAS 飙升≈ "速生桉"，BIAS 温和 ≈ "紫檀"。<!-- confidence: EXTRACTED -->
