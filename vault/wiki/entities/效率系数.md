# 效率系数

> Efficiency Ratio（ER）— [[佩里·考夫曼]] 提出的**趋势强度归一化指标**，[[考夫曼自适应均线]] 的核心调速器。

## 一句话定义

**净位移 ÷ 路径长度** — 衡量过去 N 期价格走得"直"还是走得"弯"，结果归一化到 [0, 1]，可作为通用的趋势-震荡分类器。

## 公式

```
direction  = abs(P_t - P_{t-N})
volatility = Σ_{i=t-N+1}^{t} abs(P_i - P_{i-1})

ER = direction / volatility    ∈ [0, 1]
```

直观理解：

- 如果价格从 A 直线走到 B（每步同向），direction ≈ volatility，ER ≈ 1
- 如果价格在 A、B 之间来回横盘（净位移小但路径长），ER ≈ 0

## 取值语义

| ER 取值 | 市场状态 | KAMA 反应 |
|---|---|---|
| ≥ 0.7 | 强趋势（单边市） | 灵敏，紧贴价格 |
| 0.3 - 0.7 | 弱趋势 / 转折期 | 中等平滑 |
| ≤ 0.3 | 震荡市 / 横盘 | 迟钝，过滤噪音 |

> 阈值仅供参考，A 股不同品种 / 不同周期需独立校准 <!-- confidence: INFERRED -->

## 在 KAMA 中的角色

ER 作为权重在快慢 EMA 之间做线性插值（再平方放大），决定每根 K 线的实际平滑速度：

```
SC = (ER * (FAST_SC - SLOW_SC) + SLOW_SC) ^ 2
```

详细推导见 [[考夫曼自适应均线]]。

## 独立使用价值

ER 不仅是 KAMA 内部组件，本身就是一个**可独立提取的趋势强度因子**，可作为：

- **regime 滤波器**：仅在 ER > 阈值 的样本上启用动量策略
- **仓位调节器**：仓位 = base * ER（趋势强加仓、震荡市轻仓）
- **横截面打分**：在 ETF 池内按 ER 排序，挑"走得最直"的若干只
- **与 [[均线偏离度]] 联合**：BIAS 给出"在哪"，ER 给出"走多稳"，二者交叉过滤 <!-- confidence: INFERRED -->

## 与本知识库其他趋势识别因子的对比

| 因子 | 角度 | 输出 |
|---|---|---|
| [[效率系数]]（ER） | 净位移 / 路径 | 标量 ∈ [0,1] |
| [[FIP效应]] | 趋势"光滑度" | 走势纯度 |
| [[RSRS]] | 高低点斜率 | 阻力支撑动态 |

## 潜在工程实现（V25 候选）

```python
import numpy as np

def efficiency_ratio(prices: np.ndarray, n: int = 10) -> float:
    """考夫曼效率系数；prices 为最近 n+1 根收盘价"""
    if len(prices) < n + 1:
        return np.nan
    direction = abs(prices[-1] - prices[-n-1])
    volatility = np.sum(np.abs(np.diff(prices[-n-1:])))
    return direction / volatility if volatility > 0 else 0.0
```

向量化版本（pandas）：

```python
def er_series(close: pd.Series, n: int = 10) -> pd.Series:
    direction = (close - close.shift(n)).abs()
    volatility = close.diff().abs().rolling(n).sum()
    return direction / volatility
```

## 局限性

- 对极端跳空（涨跌停）敏感，可能瞬间冲到 1，过度乐观
- N 选择敏感：N 太小（< 5）噪声大、N 太大（> 30）反应滞后
- 不能区分"上涨趋势"与"下跌趋势"——只衡量趋势强度，方向需另配指标 <!-- confidence: EXTRACTED -->
