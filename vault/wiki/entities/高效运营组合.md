# 高效运营组合

> 充分竞争行业中筛选的"成本控制 + 资产周转效率双前 30%"标的组合

## 核心逻辑

充分竞争行业中，行业整体不存在结构性垄断利润，标的需通过**成本控制能力 + 资产周转效率**获得超额收益。"用较少资产，办较多事"是这类标的的核心特征。

## 筛选条件（按优先级）

1. **先决条件**：行业属于**充分竞争行业**（[[行业格局稳态性指标|S]] ≤ 0.7 或 [[行业盈利可持续性指标|P]] ≤ 0.7），且行业内成分股数量 ≥ 10 只（保证有充分的竞争和对比样本）
2. **成本控制能力**：滚动 3 年**销售费用率、管理费用率、财务费用率**中位数，**均处于行业内前 30%**（费用率越低，排名越靠前）
3. **运营效率**：滚动 3 年**流动资产周转率、存货周转率、应收账款周转率、ROIC 中位数**，均处于行业内前 30%
4. **最终确定**：每个行业中，取上述指标**等权综合打分最高的 1 只标的**纳入组合；若当期标的超过 60 只，取综合打分前 60 只（控制组合容量）

## 三类筛选条件的语义

### 成本控制（"省钱能力"）
- 销售费用率：低 → 品牌力强或获客效率高
- 管理费用率：低 → 组织效率高
- 财务费用率：低 → 杠杆运用合理

### 运营效率（"资金周转能力"）
- 流动资产周转率：高 → 流动资产产出效率高
- 存货周转率：高 → 库存管理好（不积压）
- 应收账款周转率：高 → 收款能力强（不被客户拖款）
- ROIC：高 → 综合资本回报高（剔除杠杆影响的真实盈利）

## 为什么 ROIC 而不是 ROE？

- ROE 受财务杠杆影响 — 高负债公司 ROE 数字可虚高
- ROIC = (税后净利润) / (有息债务 + 股东权益) — 剔除资本结构差异，反映真实回报能力

## 60 只组合容量上限的含义

- 控制组合的换手率与持仓数量
- 适配中小资金量的实盘容量
- 大资金需进一步加流动性指标过滤

## 优劣势

✅ 7 个指标多角度筛选，控制盈利质量
✅ 组合容量自然受 60 只上限约束，避免过度分散
⚠ 充分竞争行业本身利润率天花板低，长期超额需依赖"内部龙头持续优于平均"
⚠ 行业内成分股 ≥ 10 只的约束在小行业失效
