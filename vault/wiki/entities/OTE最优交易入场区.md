# OTE 最优交易入场区（Optimal Trading Entry）

> [[聪明钱概念SMC]] 体系的标准入场区域 —— **斐波那契 62% - 79% 回撤区域**，在 [[接受规则]] + [[CHoCH]] 触发后用于具体执行入场

## 一句话定义

OTE = **Optimal Trading Entry** = **斐波那契 62%-79% 回撤区域**。它本身不引发反转，但当行为已经改变（[[接受规则]]）+ 结构已经确认（[[CHoCH]]）后，**回踩到 OTE 区域往往能获得反转配合**。

## 在 SMC 时序中的位置

```
[1] [[接受规则]] 触发      ← 行为先变（早期预警）
       ↓
[2] [[CHoCH]] 确认         ← 结构后确认（最终标准）
       ↓
[3] 价格回踩到 OTE 区域    ← 本概念（执行入场）
```

> 顺序很重要：**行为先变 → 结构后确认 → 入场放在最后**。

## 斐波那契回撤区间

| 比例 | 含义 |
|---|---|
| 50% | 一般回撤位 |
| **62%** | OTE 入场上沿 |
| **79%** | OTE 入场下沿 |
| 88% | 接近全回撤，反转失败概率上升 |

OTE 区域之所以是 "最优"，是因为它**深度回撤** + **未完全失守** 的双重特征：
- 深度回撤 → 风险回报比（RR）更高
- 未失守 → 趋势反转仍有效

## 重要原则

> OTE 本身**并不会引发反转**。它只在 **行为已改变 + 结构已确认** 之后才有意义。

把 OTE 当作"信号 + 信号 + 信号"链条的最后一环，不要颠倒顺序。
