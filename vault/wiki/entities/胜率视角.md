# 胜率视角

> 浙商证券债券研究团队提出的右侧交易方法论 — 强调"在趋势确认后顺势介入"

## 一句话定义

胜率视角下的择时策略**直接从市场的"结果"出发**——以价格 / 成交量为输入，聚焦**趋势形态、动量节奏、支撑压力**等价格行为信号，通过 K 线、均线、振荡指标构建右侧交易模型，**追求高胜率而非高赔率**。

## 与 [[赔率视角]] 的对比

| 维度 | [[赔率视角]]（左侧 / 系列一） | **胜率视角**（右侧 / 系列二） |
|---|---|---|
| 数据 | 宏观基本面 + 事件 | 量价 + 技术指标 |
| 频率 | 月度 / 季度宏观指标 | 日频价量 |
| 介入时点 | 趋势尚未确立（**预判**） | 趋势确立初期或中段（**确认**） |
| 优势 | 抄底逃顶、赔率高 | 胜率高、信号连续、执行清晰 |
| 劣势 | 预判错误损失重 | 滞后性强，易踏空 |
| 模型 | XGBoost 三分类 | 多子策略 + 集成融合 |

## 优势

1. **高频、实时** — 可捕捉市场微结构变化
2. **逻辑基于趋势延续 + 行为金融假设** — 胜率可观
3. **辅助交易节奏控制 + 仓位管理** — 增强执行性

## 缺陷

1. **信号滞后** — 入场点常处趋势中段甚至末端
2. **震荡期误报频繁**
3. **缺乏因果解释**
4. **对参数敏感**

## 推荐组合方式

> 将基本面左侧判断作为**大方向锚定**，叠加技术面右侧信号作为**战术执行工具** — 浙商证券原文

也即：
- 赔率视角负责"做不做"（方向）
- 胜率视角负责"何时做"（时点）
