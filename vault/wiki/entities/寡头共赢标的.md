# 寡头共赢标的

> 高壁垒稳态行业内、无绝对龙头时筛选的"势均力敌、合作共赢"标的

## 核心逻辑

高壁垒稳态行业中，若不存在绝对龙头，**几家势均力敌的寡头企业**，在能力禀赋高度相似的前提下，**更易形成合作共赢的格局**，避免价格战等囚徒困境，长期维持行业高盈利水平。

## 筛选条件（按优先级）

1. **先决条件**：[[行业格局稳态性指标|S]] > 0.7 且 [[行业盈利可持续性指标|P]] > 0.7（高壁垒稳态行业）+ 无符合条件的绝对龙头标的
2. **盈利质量**：滚动 3 年调整后**营业利润率**中位数处于行业前 30%（剔除非经常性损益）
3. **现金流质量**：滚动 3 年**经营活动净现金流/营业利润**中位数处于行业前 30%（验证盈利真实性）
4. **最终确定**：满足上述条件的标的中，取**份额、盈利质量、现金流质量等权综合打分最高的 1 只**

## 为什么这种逻辑可能成立？

- 寡头格局下：**重复博弈** → 默契合作 → 高利润均衡
- 一家独大格局下：龙头主导定价 → 第二名只能跟随 → 共赢不成立（应直接用绝对龙头逻辑）
- 完全竞争格局下：价格战 → 利润趋零（应走 [[高效运营组合]] 路径）

## 与"绝对龙头"的对比

| 选标逻辑 | 触发条件 | 选股结果 |
|---|---|---|
| 绝对龙头 | 高壁垒 + 存在份额绝对领先标的 | 1 只龙头股 |
| **寡头共赢（本页）** | 高壁垒 + **无绝对龙头** | 1 只综合打分最高的寡头股 |
| [[高效运营组合]] | 充分竞争 | 每行业 1 只 / 总数 ≤ 60 |

## 优劣势

✅ 利用行业结构性 alpha（寡头默契）
✅ 三个财务指标共同筛选，控制盈利质量
⚠ "无绝对龙头"判断未给出量化标准（可能是"市占率最高的两家差距 < 5%"，需复测时确认）
⚠ 寡头默契假设在反垄断 / 跨行业竞争冲击下可能失效
