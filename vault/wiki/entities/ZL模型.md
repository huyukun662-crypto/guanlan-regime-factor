# ZL模型

> 基于蒙特卡洛模拟 + 条款博弈最优停时的可转债结构化定价模型，是可转债估值的"防守型绝对价值锚"。

## 简介

ZL 模型（Zheng-Lin 模型）将可转债视为"信用底仓 + 多重内嵌奇异期权"的复杂结构化产品，立足发行人与投资者条款博弈的最优决策机制，运用 [[蒙特卡洛模拟]]（结合 GPU 算力加速）显式追踪 [[强赎条款]]、[[回售条款]]、[[下修条款]] 等内嵌期权的触发路径。其定价逻辑高度贴近合同现金流真实演化，估值偏向保守，是熊市与下行周期的核心防守工具。

## 关键信息

- **类型**：定价模型 / 概念
- **领域**：[[可转债]] 量化定价、事件驱动策略基础
- **数学结构**：数值解（蒙特卡洛 + 最优停时 / 动态规划）
- **常见对照**：[[BS模型]]（静态欧式期权解析解）

## 详细内容

### 六步计算流程（来源：浙商证券 2026-03-10 报告）

a) **路径模拟**：假设正股价在风险中性世界服从几何布朗运动，模拟海量股价路径。

b) **路径触发判断**：自赎回 / 回售期开始，每个计息年度连续追踪。若率先触发赎回，路径回报锁定为 $\text{Max}(S_t \times FV/X, B_c)$（$S_t$ 为此时正股价、$FV/X$ 转股比例、$B_c$ 赎回价）。

c) **回售触发处理**：若率先触发回售，模型假定发行人主动下修转股价以诱使投资者放弃回售，下修幅度设定为使转债理论价值略高于回售价。在此环节嵌入 BS 公式近似计算回售日理论价值，反算最优转股价并刷新后续路径。

d) **循环判断**：在赎回触发前对所有计息年度循环执行触发判断；全程未触发强赎则按到期真实回报贴现。

e) **平均聚合**：对所有路径贴现回报取平均值，确立理论绝对价值。

### 核心壁垒

- **显式刻画跳跃风险与极值约束**：通过路径依赖追踪条款博弈，规避欧式期权简化偏差。
- **充分计入下修预期与强赎阻断**：估值天然偏向防守，在熊市提供扎实债底支撑。
- **可延展至事件驱动策略量化增强**：[[事件驱动策略]] 的天然底座。

### 局限

- **算力成本高昂**：依赖 GPU 并行；对历史股价、转股价动态调整、条款存续状态等高频细颗粒度数据极度敏感。
- **下修幅度估计偏差**：在回售压力下使用 BS 公式反算最低下修幅度，因 BS 本身高估倾向，导致下修幅度不足，从而**系统性微幅低估**整体价值（Bias -4.73 元）。
- **牛市偏保守**：理性条款约束机制使其在估值非理性抬升的牛市中难以赚取"泡沫溢价"。

### 整体精度（2019 年至今全样本严约束样本池）

- Bias -4.73 元（轻微低估）
- MAE 11.66 元；MAPE 7.88%；SMAPE 8.14%（相对误差全面优于 [[BS模型]]）

### 策略表现

- **2019 年至今 严约束多头**：年化 15.33%，最大回撤 13.74%（低于基准 14.91%），夏普 **1.44**（高于 BS 多头的 1.35）
- **错误定价因子**（zl_deviation）：单因子年化超额 7.19%、夏普 **1.29**（全组最优）、最大回撤 -13.15%；六因子等权合成年化超额 13.08%
- **2022Q1–2024Q3 下行周期全域回测**：
  - 纯空头年化 14.78%、夏普 0.88 → 显著优于 BS 空头（8.52% / 0.66）
  - 多空组合年化 11.49%、夏普 0.90 → 全面压制 BS 多空（3.93% / 0.47）
