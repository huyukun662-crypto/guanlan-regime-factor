# 成交量相关性过滤

> RSRS 策略的另一种熊市保护：用成交量与 RSRS 修正标准分的相关性过滤左侧买入信号

## 一句话定义

只有当 [[RSRS]] 买入信号触发，且**最近 10 日成交量与 [[RSRS修正标准分]] 的相关性为正**时，才执行买入；卖出信号无条件立即执行。

## 具体规则

```
买入条件 = RSRS 买入信号触发 AND  corr(volume[-10:], z_modified[-10:]) > 0
卖出条件 = RSRS 卖出信号触发  (任何条件下)
```

## 经济逻辑

- 市场上涨通常伴随成交量放大
- 下跌时成交量往往萎缩
- 当成交量与 RSRS 信号正相关 → 市场处于"放量上涨"状态 → 信号可信
- 当成交量与 RSRS 信号负相关 → 可能是"无量上涨"（虚假反弹）或"放量下跌"中的反技术信号 → 屏蔽

## 与 [[价格趋势过滤]] 的对比

| 维度 | 价格趋势过滤（4.1） | 成交量相关性过滤（4.2） |
|---|---|---|
| 数据 | 仅价格 (MA20) | 价格 + 成交量 |
| MDD 控制 | 50%+ → 20% | 略逊 4.1 |
| 胜率 + 盈亏比 | 中等 | **更优** |
| 收益率 | **更优** | 略逊 4.1 |

## 实战建议

- 收益偏好 → 价格趋势过滤
- 胜率偏好 → 成交量相关性过滤
- 综合 → 两者都加（双重过滤，保守但更稳）
