# TL 国债期货

> 中金所 **30 年期国债期货**主连合约

## 基本信息

- **代码**：TL（主连合约）
- **标的**：30 年期国债
- **交易所**：中国金融期货交易所（CFFEX）
- **上市时间**：2023 年 4 月 21 日（较 T 晚 8 年）

## 在知识库中的应用

### [[2025-07-31-zheshang-rate-timing-winrate]] 多信号策略回测

| 指标 | 值 |
|---|---|
| 回测区间 | 2023/4/21-2025/7/29（2.3 年） |
| 年化收益 | **23.22%** |
| 累计收益 | 57.59% |
| 夏普比 | **3.78** |
| 最大回撤 | 2.28% |
| 交易次数 | 26 |
| 平均持仓 | 16 天 |
| **胜率（次）** | **80.77%** |

特征：
- 长久期 → 波动较 T 大幅放大（2-3 倍）
- **2025 年 2 月 + 7 月**两次大幅回撤，多信号策略主动避开 → 防御能力体现
- 由于 J 值（KDJ）常出现极端跳动，传统单一指标在 TL 上稳定性不足
- 假突破较多 → 必须用多信号融合策略

## 与 [[T国债期货]] 的对比

| 维度 | TL (30 年) | T (10 年) |
|---|---|---|
| 久期 | 长 | 中等 |
| 波动率 | 较高 | 较低 |
| 上市时间 | 2023 | 2015 |
| 多信号策略年化 | **23.22%** | 8.63% |
| 多信号夏普 | **3.78** | 3.04 |
| 样本期信度 | 短，置信区间宽 | 长，更可信 |

## 风险与待验证

- 样本期仅 2.3 年，年化 23.22% 的置信区间宽
- 上线时间晚 → 历史压力测试不足（缺 2015 / 2018 / 2022 年大波动期数据）
