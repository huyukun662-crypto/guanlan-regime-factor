# IC 选股策略

> 用因子（或模型预测值）截面值排序选股的标准量化范式

## 一句话定义

每个交易日（或调仓日），用因子值或模型预测值在**横截面**上对所有候选股票排序，**取 Top-N 等权或加权构建多头组合**，定期调仓。

## 标准流程（OpenAlphas LightGBM 笔记的实现）

```
每日收盘后:
1. 用最新因子数据输入模型，得到全市场 5 日收益率预测值 ŷ
2. 剔除不可交易的股票（ST / 涨跌停 / 停牌 / 新股）
3. 选预测值最高的 50 只 → 等权构建持仓
4. 每 5 个交易日调仓一次
```

## 关键参数

| 参数 | 典型值 | 影响 |
|---|---|---|
| Top-N 持仓数 | 30-100（OpenAlphas 用 50） | 越大越分散，alpha 衰减 |
| 调仓周期 | 5-20 日（OpenAlphas 用 5 日） | 越短换手成本越高 |
| 加权方式 | 等权 / 预测值加权 | 等权降低头部股权重 |
| 风控约束 | 行业 / 风格中性 / 个股权重上限 | 防过度集中 |

## 容量与换手率的权衡

- **持仓数少 + 短调仓**：高 alpha，但容量小、换手成本高
- **持仓数多 + 长调仓**：低 alpha，但容量大、可承载大资金

## 在素材中的出现

- [[2026-05-05-openalphas-lightgbm-bayesian]]：50 只等权 + 5 日调仓 = 周频中等容量策略
- [[2026-05-05-openalphas-alphazero-factor-mining]]：多空组合（多/空各 Top-30 - 隐含）
- [[2026-04-23-openalphas-bottom-style-timing]]：风格切换型组合（不算典型 IC 选股）

## 与其他选股范式的关联

- vs [[价值投资量化框架]]：基本面财务质量筛选 → 1 只/行业 → 月/季频
- vs [[寡头共赢标的]]：基本面 + 行业内排序 → 单一标的 → 季频
- 本范式：**纯量化预测 + 截面排序 + Top-N 等权 → 周/月频**
