# QuantsPlaybook

> 国内中文量化界最完整的开源策略复现仓库之一 — github.com/hugo2046/QuantsPlaybook

## 基本信息

- **作者**：[[hugo2046]]（shen.lan123@gmail.com）
- **URL**：[github.com/hugo2046/QuantsPlaybook](https://github.com/hugo2046/QuantsPlaybook)
- **默认分支**：`master`
- **最近更新**：2025-12-14（持续维护）
- **核心定位**：用 Python 系统性复现国内券商金工研报

## 数据依赖

- **聚宽 JQData**（需 API Key，付费）
- **Tushare Pro**（需 API Key，部分免费）
- 知识库推荐改用 [[akshare]]（无需 API Key）

## 与本知识库的角色定位

这个仓库是知识库**理论 ↔ 代码**的桥梁：
- 已消化的理论素材（OpenAlphas / 浙商 / 广发 / 微信公众号）→ 在仓库找起点代码
- 仓库未在知识库覆盖的策略 → 可作为新一批 ingest 对象
