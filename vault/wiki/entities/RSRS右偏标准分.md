# RSRS 右偏标准分

> RSRS 三层标准化体系的最高级：相对位置 × 拟合可靠性 × 绝对强度

## 公式

```
z_t^右偏 = z_t · R_t² · β_t
       = 相对位置 × 拟合可靠性 × 绝对强度
```

## 三要素语义

| 因子 | 语义 | 含义 |
|---|---|---|
| `z_t` | 相对位置 | β 在历史分布中的百分位 |
| `R_t²` | 拟合可靠性 | OLS 拟合越好，可靠性越高 |
| `β_t` | 绝对强度 | β 本身的幅值（支撑/阻力相对强度的绝对值） |

## 为什么叫"右偏"？

由于 [[RSRS]] 斜率 β 几乎总为正数，而修正标准分本身分布近似对称，相乘后**整体分布向右偏**（向"强支撑信号"侧）。

这种特性使其更适合**多头偏好**策略：
- 极端右偏值 → 强烈看多 → 触发买入
- 中性附近 → 不操作
- 极端左偏值（罕见）→ 极强看空 → 触发卖出

## 实战阈值

OpenAlphas 笔记 fig14 代码使用 `threshold=0.7`：
- z^右偏 > 0.7 → 买入
- z^右偏 < -0.7 → 卖出
