# 策略衰减风险（Strategy-Decay Risk）

> [[Frank Fabozzi]] 团队论文核心概念 —— **"策略本身失去有效性"** 作为量化投资的**元风险**（Meta Risk），区别于传统的收益波动风险

## 一句话定义

不是某次大回撤的瞬时风险，而是 **策略的 Sharpe / IC / Alpha 在时间维度上慢慢消失** 的结构性风险。**最危险的不是亏钱，是策略开始失效而你尚未察觉**。

## 与传统风险的根本区别

| 维度 | 传统风险 | 策略衰减风险 |
|---|---|---|
| 表现 | 单期大幅亏损 | 长期慢慢失效 |
| 衡量 | 波动率 / MDD / VaR | [[MRP最小Regime表现]] |
| 时间尺度 | 日 / 月 | 年 / 多周期 |
| 可见性 | 立即可见 | **难以察觉** |
| 救援方式 | 止损 / 减仓 | 策略**报废** |

## 现象：慢慢失效

> "很多策略并不是因为一次大回撤死亡，而是慢慢失效。Sharpe 不断下降；IC 不断衰减；Alpha 逐渐消失。最终沦为'历史上有效'。"

2021 年以来的典型案例：
- 2024 年 1 月 Size 因子失效
- Value 因子长期低迷
- 反转策略衰退
- 部分机器学习 Alpha 快速拥挤

## 论文论点：传统风险模型遗漏了关键问题

| 问题 | 传统模型回答 |
|---|---|
| "当策略**有效**时，会亏多少？" | ✅ 波动率 / VaR / ES |
| "当策略**失效**时，会发生什么？" | ❌ 无 |

→ **策略衰减风险就是这个被遗漏的维度**。

## AI 时代会更严重

> AI 策略的衰减速度可能**远高于传统因子**。
> 原因：AI 模型学习的是市场中的**信息结构**，而信息结构变化速度**远快于基本面因子**。

→ AI 时代的量化最大风险，不是模型误差，而是 **模型寿命**。

## 衡量工具：[[MRP最小Regime表现]]

```
MRP = Min(Sharpe across regimes)
```

把历史切分为多个市场状态 → 取最差 Sharpe → 越低越说明策略对 regime 切换越脆弱。

## 与 V25 项目工程纪律的呼应

V25 ETF2.0 v5i 复盘心得（来自 CLAUDE.md）已经在工程层面**预防**了策略衰减风险：

- "**IS/OOS 纪律是硬约束**" → 不让"IS Sharpe 漂亮但 OOS 崩溃"的策略上线
- "**单变量迭代原则**" → 不让多变量混淆下的过拟合策略意外通过
- "**Pareto 优化 > 单目标优化**" → "无害且偶尔有效"也是胜利 → 重视稳定性

→ 工程纪律是**事前预防**，MRP 是**事后量化**。<!-- confidence: INFERRED -->
