# SAR 指标

> Stop and Reverse — 抛物线转向指标，威尔斯·威尔德发明的**趋势 + 动态止损**双功能经典技术指标

## 一句话定义

用**抛物线轨迹追赶价格**：上升趋势中 SAR 点画在 K 线下方（绿点），下降趋势画在上方（红点）；价格穿透 SAR 点 → 趋势反转 + 触发止损。"画一条移动止损护卫线"。

## 公式

### 上涨趋势 SAR（绿点）
```
SAR_n = SAR_{n-1} + AF × (EP - SAR_{n-1})
```

### 下跌趋势 SAR（红点）
```
SAR_n = SAR_{n-1} - AF × (SAR_{n-1} - EP)
```

### 关键参数

| 参数 | 默认 | 说明 |
|---|---|---|
| **AF（加速因子）** | 初始 0.02，每次 +0.02，上限 0.2 | 价格创新高/新低时增加 → SAR 点加速追上价格 |
| **EP（极值点）** | 当前趋势内最高/最低价 | 趋势内不断更新 |
| 起步 SAR | 近期最低价（多）/ 最高价（空） | 趋势刚启动时设定 |

## 为什么叫"抛物线"

- AF 从 0.02 起步 → 缓慢追赶
- 每次创新高 +0.02 → 加速度递增
- 直到上限 0.2 → 加速度饱和
- ⇒ SAR 点轨迹呈**抛物线形态**（向下凹的反曲线）

## 三大用法

### 1. 趋势跟踪（持仓信号）
- SAR 在 K 线**下方** → 多头 → 持有/加仓
- SAR 在 K 线**上方** → 空头 → 空仓/做空
- 点距扩大 → 趋势加速；点距走平 → 趋势衰竭

### 2. 动态止损（核心价值）
- 价格跌破 SAR 点即止损（trailing stop）
- 短线：SAR 点下方 1% 作为保护性止盈
- 长线：结合周线 SAR 锁定大利润

### 3. 趋势反转
- 多转空：股价从上方跌破 SAR → 卖出 + 反手做空
- 空转多：股价从下方升破 SAR → 买入 + 反手做多

## 与同类指标的对比

| 指标 | 类型 | 核心功能 | 致命弱点 |
|---|---|---|---|
| **SAR（本指标）** | 趋势 + 止损 | 双功能：方向 + 止损位 | 震荡市频繁转向；缺口跳空失灵 |
| [[均线偏离度]] | 趋势强弱 | 阈值过滤主线 | 不直接给止损位 |
| [[RSRS]] | 阻力支撑相对强度 | 高低价 OLS 斜率 | 左侧抄底倾向 |
| [[MACD]] | 趋势确认 | 双线动量差 | 滞后明显 |
| [[KDJ]] | 震荡指标 | 超买超卖 | 趋势市频繁误报 |
| [[鳄鱼线]] | 三均线状态 | 多/空/沉睡 三态判断 | 平移引入额外滞后 |
| [[九转序列]] | 结构衰竭 | TD Sequential 计数 | 信号稀疏 |

⇒ **SAR 的独特性**：**趋势 + 止损一体**，无需另外维护 stop loss 逻辑。

## 实战增强技巧

### 参数优化（不同行情）
| 行情 | AF 初始 | AF 上限 | 配套指标 |
|---|---|---|---|
| 短线波段 | 0.04 | 0.2 | [[MACD]](5,13,1) 金叉 |
| 中线趋势 | 0.02 | 0.1 | 60 日均线同向 |
| 极端行情 | 0.2（固定） | 0.2 | ATR(14) 波动幅度过滤 |

### 多周期共振
- **日线 + 周线 SAR 同向** = 大级别趋势确认
- **小时线 SAR 转向** = 短期变盘预警

## 优势

1. **傻瓜式操作**：红绿点清晰指示方向，对新手友好
2. **动态风控**：trailing stop 自动锁利润
3. **单边行情捕捉力强**：主升 / 主跌浪都能跟上

## 致命弱点

1. **震荡市频繁转向**：横盘时 SAR 反复跳变 → 需 [[KDJ]] / RSI 超买超卖过滤
2. **加速因子滞后**：AF 从 0.02 起步 → 急速行情初期可能错过
3. **缺口跳空失灵**：财报暴雷 / 重大事件穿透 SAR 点
4. **不适合震荡市**：与 [[KDJ]] 形成互补关系（一个适合趋势，一个适合震荡）

## 历史溯源

- **作者**：威尔斯·威尔德（J. Welles Wilder Jr.）
- **同一作者发明的其他指标**：RSI（相对强弱指标）、ATR（平均真实波幅）、ADX（平均方向指数）
- **原始论文 / 著作**：《New Concepts in Technical Trading Systems》（1978）
