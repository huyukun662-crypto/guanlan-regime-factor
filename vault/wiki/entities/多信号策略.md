# 多信号策略

> 浙商证券[[胜率视角]]下的核心集成方法 — "**任意 2 个子策略发出买入信号即视为入场**"

## 一句话定义

把 5 个子策略（[[低延时趋势]] / [[阶矩择时]] / [[单/双均线策略]] / [[通道过滤]] / [[九转序列]]）的最优参数信号汇总，用**"交叉检验信号触发法"**做集成 — 类似机器学习的 **ensemble 模型**。

## 持仓规则

```
若 任意 2 个子策略发出买入信号  →  入场（持多）
若 无信号                       →  空仓 / 平仓
```

## 三层结构（"趋势识别 - 节奏控制 - 信号整合"）

| 层 | 功能 | 子策略 |
|---|---|---|
| **趋势识别** | 找方向 | [[低延时趋势]]、[[单/双均线策略]] |
| **节奏控制 / 波动结构** | 找时机 | [[阶矩择时]]、[[通道过滤]] |
| **结构衰竭 / 信号整合** | 找拐点 | [[九转序列]] |

## 集成的核心价值

1. **将子策略优势相结合** — 每个子策略各擅其所
2. **多信号交叉检验** — 降低单策略误报率
3. **保持信号稳定性** — 减少频繁切换
4. **降低交易频率** — 节约成本（避免被滑点吃光）

## 回测结果

| 指标 | T (10 年国债期货) | TL (30 年国债期货) |
|---|---|---|
| 回测期 | 2015/3/20-2025/7/29 (10 年) | 2023/4/21-2025/7/29 (2.3 年) |
| 年化收益 | 8.63% | **23.22%** |
| 累计收益 | 128.72% | 57.59% |
| 夏普比 | 3.04 | **3.78** |
| 最大回撤 | 2.62% | 2.28% |
| 交易次数 | 108 | 26 |
| 平均持仓天数 | 16 | 16 |
| **胜率（次）** | 70.37% | **80.77%** |
| 平均盈亏比（天） | 1.59 | 1.87 |

⚠ 不考虑交易滑点 / 杠杆 / 保证金。

## 极端行情防御能力（TL 案例）

- 2025 年 2 月 + 7 月利率快速上行期 → 多信号策略**主动避开**
- **趋势识别 + 动量过滤 + 结构确认** 共同作用 → 及时避免错误入场
- 净值保持稳定，规避下行风险

## 后续优化（作者列出 3 项）

1. **信号权重机制** — 趋势类（低延时 / 双均线）vs 结构类（九转 / 布林带）分层加权（线性 / 逻辑门）
2. **数据维度扩展** — 引入新闻文本 / 政策文本 → LLM（GPT 微调 / DeepSeek）情感打分
3. **集成层升级** — 从规则法则 → Stacking / Voting Ensemble（弱监督学习）

## 与其他集成方法的对比

| 方法 | 输入 | 集成方式 |
|---|---|---|
| **本策略（任意 2 信号）** | 5 子策略二值信号 | 触发计数（≥2） |
| Voting Ensemble | 模型概率输出 | 多数投票 |
| Stacking | 第一层模型输出 | 第二层 meta-learner |
| 信号权重融合 | 子策略信号 + 权重 | 线性加权 |
