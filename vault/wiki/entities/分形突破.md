# 分形突破

> Fractal Breakout — 比尔·威廉斯基于分形几何思想构造的"5 根 K 线突破"信号

## 一句话定义

**上分形** = 至少 5 根连续 K 线，最高点在中间，左右各有 2 根**较低的高点**。
**下分形** = 对称定义。

收盘价**突破最近上分形高点 → 看多**；**跌破最近下分形低点 → 看空**。

## 学术源头

- **分形（Fractal）** 概念由波兰数学家 Mandelbrot 提出
- 在金融市场中：市场总会向某方向运动 → 多头入场 → 抛压 → 回调 → 新信息 → 再涨；如果突破前期向上分形则强势
- 比尔·威廉斯在《证券混沌操作法》中引入

## 形态识别

```
5 根 K 线（中间为最高 / 最低）：
       ↑（中指）
   /       \
  /         \
 ←  →       ←  →
  最高点高于 / 最低点低于左右各 2 根 K 线

形态 A: 标准分形
形态 B: 同时是上下分形
形态 C: 6 根 K 线共用价格柱
形态 D: 第 5 根 K 线最高点 = 此前最高点
```

## 信号规则

| 条件 | 信号 |
|---|---|
| T 日收盘 > 最近上分形高点 | 看多（满仓）|
| T 日收盘 < 最近下分形低点 | 看空（空仓）|
| 其他 | 维持前一日仓位 |
