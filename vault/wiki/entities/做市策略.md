# 做市策略

> Market Making — **同时提交买单和卖单**赚取价差，向市场提供流动性

## 一句话定义

做市商**同时挂出买价（bid）+ 卖价（ask）**，赚 spread 收益。但要面对两大风险：**库存风险**（持仓被市场逆向冲击）+ **逆向选择风险**（被知情交易者吃单后行情真转向）。最优做市 = 在**报价价差** + **报价宽度** 之间动态平衡。

## 经济类比（来自本书 7.1）

> "做市商类似 24 小时便利店"
> - 农民（卖方）随时可以送来 100 加仑牛奶
> - 消费者（买方）随时可以买到牛奶
> - 便利店（做市商）赚"差价"（spread）作为提供流动性的报酬

如果附近有很多农场（高竞争），便利店利润空间小；如果牛奶变质快（高波动），便利店要更频繁调价。

## 经典模型 1：Ho-Stoll 1981

> Ho-Stoll 是**做市开山论文**，提出"库存风险驱动的最优报价"框架。

### 核心思想
- 做市商有库存 q
- 库存 q 偏正（持多）→ 倾向**降低 ask** 卖出 / 提高 bid 不再买
- 库存 q 偏负（持空）→ 反向

### 最优报价
```
ask_price = mid + spread/2 + γ·q
bid_price = mid - spread/2 + γ·q
```
（γ = 库存风险厌恶系数）

⇒ 库存 q 越偏离 0，整个报价向相反方向偏移。

## 经典模型 2：Avellaneda-Stoikov 2008

> 现代做市策略的事实标准 — 把 Ho-Stoll 推广到**有限期 / 随机价格 / mean-variance 效用**。

### 关键贡献

定义两个核心参数：
- **保留价**（Reservation Price）= 真实"心理报价"
- **报价价差** = 围绕保留价的对称偏移

```
r(s, q, t) = s - q·γ·σ²·(T - t)             # 保留价
spread(t) = γ·σ²·(T - t) + (2/γ)·ln(1 + γ/k)  # 报价价差
```
- s = 中间价
- q = 库存
- γ = 风险厌恶
- σ² = 价格波动率
- T - t = 剩余时间
- k = 订单到达速率参数

### 直觉
- **库存 q ↑ → 保留价 ↓**（想出货）
- **波动 σ ↑ → 价差 ↑**（保护自己）
- **临近收盘 t→T → 价差收窄**（赶紧平仓）
- **订单流密集 k ↑ → 价差收窄**（市场流动性好，可窄报价）

## 高级扩展（本书 7.3）

### 脉冲控制（Impulse Control）
处理"何时打掉对手报价（市价单）"的问题，与连续控制（限价单）混合。

### 智能订单路径 + 暗池（7.4-7.6）
- **SOR**（Smart Order Routing）：跨多个交易所最优分单
- **暗池**：非公开 LOB，做市商需要"删失探索"（censored exploration）— 因为暗池只告诉你成交了多少，不告诉你需求多少

## 关键风险

### 1. 库存风险（Inventory Risk）
做市商不希望长期持仓 → 持仓越多越急于平仓 → 报价偏移
**对策**：[[Avellaneda-Stoikov模型]] 的 q 项

### 2. 逆向选择风险（Adverse Selection）
对手是**知情交易者**（HFT / 内幕信息）→ 你的报价被打掉后行情确实朝那个方向走 → 亏损
**对策**：识别订单流毒性（toxicity）+ 撤单 / 加宽价差

### 3. 竞争风险
其他做市商竞争同一价位 → 价差被压缩到极限
**对策**：速度（毫秒级 → 微秒级 → 纳秒级）

## 与已有素材的关联

### 与 [[最优执行]] 的对偶性
- 最优执行 = **吃单一方**的优化（消耗流动性）
- 做市 = **挂单一方**的优化（提供流动性）
- 两者**博弈对偶**：做市商必须考虑被最优执行算法打的概率

### 与 [[限价订单簿LOB]]
- 做市的**所有信号源**都来自 LOB（队列长度、不平衡、订单到达率）
- 现代做市基本都用 [[Hawkes过程]] 建模订单流

### 与本知识库其他主题的距离
- 知识库目前**完全无做市素材** — 本主题填补 100% 空白
- V25 项目目前不做市（只做投资 / 投机），但理解做市策略有助于估计**滑点和市场冲击**

## 优势 / 劣势

✅ 现金流稳定（每日吃 spread）
✅ 不需要预测方向，只需要管理库存
✅ 是交易所"刚需"角色

⚠ 进入门槛极高（需要顶级基础设施 + 数据）
⚠ 大单冲击 / 闪崩瞬间损失巨大
⚠ A 股做市机会有限（科创板 / 期权做市）
