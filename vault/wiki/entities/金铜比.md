# 金铜比

> 黄金价格 / 铜价的比值，作为**增长压力 / 衰退预期的 regime 指标**——铜是工业需求（增长）代理，黄金是避险代理，比值上行 = 增长担忧压过工业需求。

## 简介

[[滞胀交易三阶段框架]] 中区分阶段一与阶段二的核心可观测信号。铜价反映工业需求强弱（"博士铜"），黄金反映避险与流动性。**金铜比稳定 → 增长韧性仍在（阶段一）；金铜比快速上升并维持高位 → 衰退交易接管（阶段二）。**

## 关键信息

- **类型**：概念 / 宏观 regime 指标
- **领域**：大类资产、宏观择时
- **可量化形式**：`gold_price / copper_price`（或反向看铜金比 `copper/gold`），常用滚动 z-score 或同比变化判断方向
- **数据源**：LME 铜价、COMEX 黄金（可经 akshare / yfinance 获取）
- **相关概念**：[[滞胀交易三阶段框架]]、[[期限利差倒挂]]

## 详细内容

历史信号（出自华泰策略复盘）：
- **70 年代后期阶段一**（1978/10–1979/9）：铜价 +45%、黄金 +47%，**金铜比维持稳定**，反映工业需求预期仍具韧性，市场对增长信心未动摇。
- **70 年代后期阶段二**（1979/9 起）：美股 -27%、铜等工业金属同步重挫，**金铜比快速上升并维持高位**——衰退交易的典型特征。
- **当前**：铜与黄金比价从 3 月初至今提升超 10%（即金铜比下降），华泰据此判断市场"部分但未完全定价增长下行"——增长下修定价仍不充分。

## 与自有策略的接口

- 候选用法：金铜比上行加速（z-score 突破阈值）→ 触发权益降权 / 切防御篮子的 regime 门控信号之一。
- 与库内既有防御机制（[[BIAS减仓]] 事后干预、[[RSRS]] 事前择时）属**不同信息源**（宏观 vs 价量），可做多信号叠加。
