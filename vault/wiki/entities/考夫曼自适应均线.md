# 考夫曼自适应均线

> 又名：AMA / KAMA / Adaptive Moving Average — 由 [[佩里·考夫曼]]（Perry Kaufman）提出的**自适应平滑均线**，通过 [[效率系数]]（ER）让平滑速度随市场状态动态切换。

## 一句话定义

**根据当前市场是趋势还是震荡，自动调整 EMA 平滑因子**——趋势市变快（紧咬价格）、震荡市变慢（过滤噪音）。是固定窗口 [[SMA]] / [[EMA]] 在"窗口长度二选一"困境下的结构性改进。

## 核心机制

### 三步公式

1. **算 [[效率系数]] ER**（市场状态分类器）

   ```
   direction = abs(P_t - P_{t-n})
   volatility = sum(abs(P_i - P_{i-1}) for i in last n)
   ER = direction / volatility    ∈ [0, 1]
   ```

   - ER → 1：方向性强（趋势）
   - ER → 0：来回乱窜（震荡）

2. **算平滑常数 SC**（ER 加权混合快慢 EMA）

   ```
   SC = (ER * (FAST_SC - SLOW_SC) + SLOW_SC) ^ 2
   FAST_SC = 2 / (2 + 1)  = 2/3      # 快速 EMA 周期 = 2
   SLOW_SC = 2 / (30 + 1) = 2/31     # 慢速 EMA 周期 = 30
   ```

3. **递推更新 AMA**

   ```
   AMA_t = AMA_{t-1} + SC * (P_t - AMA_{t-1})
   ```

### 行为对比

| 市场状态 | ER | AMA 表现 | 等价于 |
|---|---|---|---|
| 强趋势 | ≈ 1 | 紧咬价格 | 接近 EMA(2) |
| 震荡市 | ≈ 0 | 几乎不动 | 接近 EMA(30) |
| 中间状态 | 0.3-0.7 | 渐变 | 介于两者 |

## 完整 AMA 系统（引擎 + 变速箱）

考夫曼原版交易系统不止一条均线，还包含 [[标准差过滤器]]：

> 只有当 `abs(AMA_t - AMA_{t-1})` 超过近 N 期 AMA 变化的 `k * σ` 时，才触发交易信号。

- **AMA = 引擎**：自适应平滑
- **过滤器 = 变速箱**：去毛刺、去假突破
- 缺失参数 `k` 需独立寻优 <!-- confidence: AMBIGUOUS -->

## 与其他均线类工具的关系

| 工具                | 关注点          | 类型    |
| ----------------- | ------------ | ----- |
| [[SMA]] / [[EMA]] | 固定窗口平滑       | 位置滤波  |
| [[考夫曼自适应均线]]      | 动态平滑（自适应窗口）  | 速度滤波  |
| [[均线偏离度]]（BIAS）   | 价格相对均线的偏离百分比 | 位置型择时 |
| [[FIP效应]]         | 趋势"形态光滑度"    | 趋势质量  |

**正交性提示**：[[均线偏离度]] 衡量"现在偏多远"，KAMA 衡量"现在走多稳"，两者属于均线择时的两个正交维度，理论上可以叠加 <!-- confidence: INFERRED -->

## 聚宽 / TA-Lib 调用

```python
import talib
ama = talib.KAMA(closes, timeperiod=30)
# 默认 fast_period=2, slow_period=30（考夫曼原版）
```

TA-Lib 只暴露 `timeperiod` 一个外部参数，fast/slow 内部写死为 2/30。如需自定义需手撕公式。

## 实战注意事项

1. **TA-Lib 实现是 KAMA**：`talib.KAMA()` 函数即考夫曼版本，timeperiod=30 是 ER 窗口（不是 EMA 周期）<!-- confidence: EXTRACTED -->
2. **务必配过滤器**：裸 KAMA 在震荡市仍会发假信号，[[标准差过滤器]] 不是可选项
3. **A 股适用性需独立验证**：原书基于美股 / 期货，A 股 T+1 + 涨跌停场景下的表现需在本地 backtester 验证 <!-- confidence: INFERRED -->
4. **不是圣杯**：原文反复强调"AMA 比普通均线聪明，但不是万能圣杯"

## 在 V25 ETF 项目中的潜在位置

候选机制（A 档，尚未纳入 grid_search）：

- **趋势门控**：用 ER 替代 / 叠加现有 MA50 信号，在 ER < 0.3 时减仓到防御篮子
- **快慢 KAMA 交叉**：作为 ETF 选股的辅助过滤层，配合现有动量排序
- **过滤器 σ 倍数**：作为 grid_search 一维新参数扫描（候选区间 1.0-2.5 σ）

## 局限性

- 单标的择时工具，未天然适配横截面 ETF 轮动
- ER 对极端跳空（如涨跌停）敏感，可能高估趋势强度
- 与已经存在 [[均线偏离度]] 的信号叠加时，需先做相关性检查避免重复曝险 <!-- confidence: INFERRED -->
