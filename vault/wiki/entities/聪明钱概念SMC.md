# 聪明钱概念（Smart Money Concepts, SMC）

> 外汇 / 加密圈流行的**价格行为分析体系** —— 通过外部高低点、市场结构转换（[[CHoCH]]）、订单块、流动性区域等概念，模仿"机构资金"的交易决策逻辑

## 一句话定义

SMC 是一套**完整的价格行为分析体系**，核心假设是**机构资金（"聪明钱"）会在散户密集挂单的位置反向操作**；通过识别这些"机构印记"（结构突破、流动性扫盘、订单块吸筹等）来跟随机构方向。

## 核心构件

| 构件 | 含义 |
|---|---|
| **市场结构** | 外部高点 + 外部低点序列 |
| [[CHoCH]] | Change of Character，结构转换确认 |
| BOS | Break of Structure，结构延续突破（与 CHoCH 相对） |
| Order Block | 订单块，机构吸筹/派发区域 |
| Liquidity Zone | 流动性区域，散户止损密集区 |
| FVG | Fair Value Gap，价差缺口 |
| [[OTE最优交易入场区]] | 斐波那契 62%-79% 回撤入场点 |
| [[分布通道]] | 围绕中轴的带状区间（本知识库 2026-05-11 来源） |
| [[接受规则]] | 多 K 线连续收盘越过通道（早期预警） |

## 流派特点

- ✅ 强调**价格行为本身**，不依赖任何指标
- ✅ 体系完整，从结构识别到入场出场都有规则
- ✅ 适合外汇 24h 连续交易
- ⚠ 主观成分大（外部高/低点的"显著性"判断）
- ⚠ 无统计回测传统，多为图表案例归纳
- ⚠ A 股因涨跌停 + T+1 制度，**直接套用需重大改造**
