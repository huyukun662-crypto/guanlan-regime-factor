# BS模型

> 基于 Black-Scholes 无风险套利框架的可转债静态解析定价模型，是可转债估值的"进攻型定价锚"。

## 简介

BS 模型（Black-Scholes 模型）将可转债价值静态解构为"纯债现值 + 标准欧式看涨期权"的线性组合，是中国可转债市场最经典的相对估值工具之一。其优点是形式简洁、计算并发效率极高、能输出 Delta/Gamma/Vega 等完备希腊字母，适合大规模实时定价与对冲风控；缺点是忽略 [[强赎条款]]、[[下修条款]]、[[回售条款]] 等内嵌期权的路径依赖，易在强股性单边上行市中系统性高估。

## 关键信息

- **类型**：定价模型 / 概念
- **领域**：[[可转债]] 量化定价、固收 +、期权定价
- **数学结构**：闭式解析解（continuous-time、Geometric Brownian Motion）
- **常见对照**：[[ZL模型]]（蒙特卡洛 + 条款博弈）

## 详细内容

### 核心定价公式（来源：浙商证券 2026-03-10 报告）

$$V_{CB} = V_{bond} + V_{option}$$

- 纯债现值：

  $$V_{bond} = \sum_{t=1}^{T} \frac{C_t}{(1+r_d)^t} + \frac{P}{(1+r_d)^T}$$

- 转股权价值（Black-Scholes 公式）：

  $$V_{option} = \gamma [S e^{-qT} N(d_1) - X e^{-r_f T} N(d_2)]$$

  $$d_1 = \frac{\ln(S/X) + (r_f - q + \sigma^2/2) T}{\sigma \sqrt{T}}, \quad d_2 = d_1 - \sigma \sqrt{T}$$

  其中 γ 为转股比例，S 正股价格，X 转股价格，$r_f$ 无风险利率，q 正股连续股息率，σ 正股年化波动率，T 剩余期限，N 为标准正态累积分布函数。

### 实战特征

- **优势**：闭式解 → 高并发；可一阶 / 二阶求导得到完整希腊字母矩阵；对正股趋势与隐含波动率极度敏锐。
- **核心局限**：未刻画强赎天花板与下修条款的路径依赖；在强股性上行市中系统性高估，尤其是深度实值（[[在值程度]] 高）的转债。
- **错误定价偏离规律**（实证）：
  - 长久期（剩余期限 > 1400 天）由于常数波动率假设外推 → BS 理论价上穿市场价
  - 中短端（200–1200 天）→ BS 理论价持续低于市场价（市场短期投机溢价 BS 无法涵盖）
  - 极短端（30–50 天）→ 时间价值快速衰减，BS 价格向市场收敛
- **整体精度**（2019 年至今全样本严约束样本池）：
  - Bias 2.13 元（系统性高估）
  - MAE 13.74 元；MAPE 9.79%；SMAPE 9.66%

### 策略表现

- **2019 年至今 严约束多头**：年化 19.30%，最大回撤 22.26%，夏普 1.35（详见 [[2026-03-10-浙商证券-可转债定价模型比较研究]] 表 3）
- **错误定价因子**（bs_deviation）单因子年化超额 11.50%、夏普 1.24；六因子等权合成年化超额 13.92%
- **2022Q1–2024Q3 下行周期全域回测**：BS 多头年化 -5.35%、夏普 -0.45 → 进攻属性在熊市完全失效
