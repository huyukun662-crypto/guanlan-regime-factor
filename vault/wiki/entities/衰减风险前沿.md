# 衰减风险前沿（Decay-Risk Frontier）

> [[Frank Fabozzi]] 论文构建的二维优化框架 —— **Sharpe Ratio × [[MRP最小Regime表现]]** 的帕累托前沿，类比 Markowitz 均值方差前沿但优化的是 **收益 × 寿命**

## 一句话定义

```
横轴: Sharpe Ratio        ← 收益（传统）
纵轴: MRP (Min Regime Sharpe) ← 寿命（新）
```

在 (Sharpe, MRP) 平面上画出所有策略 → 取**右上方非劣解**集合 → 即衰减风险前沿。

## 与 Markowitz 框架的类比

| 维度 | Markowitz 均值方差前沿 | Decay-Risk Frontier |
|---|---|---|
| 横轴 | 期望收益 | Sharpe Ratio |
| 纵轴 | 风险（- 方差） | MRP（寿命） |
| 优化目标 | **收益 × 风险** | **收益 × 寿命** |
| 提出时间 | 1952 | 2024+（Fabozzi） |

## 实例（论文实证）

经典因子在前沿上的相对位置：

| 因子 | Sharpe | MRP | 前沿位置 |
|---|---|---|---|
| Debt Issuance | 1.14 | -0.05 | 右下（高收益短命） |
| Investment | 0.45 | -1.01 | 左下（差） |
| Size | 0.16 | -0.93 | 左下（差） |
| 理想策略 | 高 | 高 | **右上**（高收益长寿） |

→ **历史最赚钱的因子未必在前沿上**。

## V25 项目升级路径

V25 v5i 复盘心得已有 **Pareto 优化 > 单目标优化** 思想（Sharpe / DD / Ann 多目标严格非劣解）。
Fabozzi 论文增加了 **MRP 维度**，把 V25 升级为 **4 目标 Pareto**：

```
Pareto Frontier(Sharpe, DD, Ann, MRP)
```

每个新参数必须**至少 1 维严格更好且其余非劣** 才入选。
