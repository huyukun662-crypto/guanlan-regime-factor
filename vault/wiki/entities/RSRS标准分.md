# RSRS 标准分

> RSRS 三层标准化体系的第一层：β 值在历史分布中的相对位置

## 公式

```
z_t = (β_t - μ_β) / σ_β
```

其中 `β_t` 是 [[RSRS]] 基础斜率值，μ_β 和 σ_β 分别是历史 β 的均值和标准差。

## 解决什么问题

- 基础 β 斜率指标的固定阈值（"β > 1 看多、β < 1 看空"）**无法适应市场环境的变化**
- 标准分把 β 转换为相对位置：当前 β 在历史分布中处于什么百分位
- 用动态阈值替代固定阈值

## 阈值用法

- 当 z_t > 阈值（例如 0.7）→ 买入
- 当 z_t < 负阈值（例如 -0.7）→ 卖出

## 派生：[[RSRS修正标准分]]

```
z_t^修正 = z_t · R_t²
```

引入 OLS 拟合的决定系数 R²，降低拟合差的标准分对策略的影响。

## 派生：[[RSRS右偏标准分]]

```
z_t^右偏 = z_t · R_t² · β_t
```

= 相对位置 × 拟合可靠性 × 绝对强度。由于 β 几乎总为正，分布右偏，更适合多头偏好的策略。
