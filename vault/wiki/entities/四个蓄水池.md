# 四个蓄水池

> [[广发证券策略团队]] 提出的资金面分析框架

## 四个蓄水池

| 蓄水池 | 代表资金 | 当前（2025/9）状态 |
|---|---|---|
| **监管资金** | 宽基股票型 ETF（科创50 / 沪深300 等） | 多周流出 → "保存实力"，可能在主动调节涨速 |
| **保险等绝对收益** | 年金、专户、保险、私募 | 加仓中 |
| **居民部门** | 杠杆 + 基金 + 直接开户 | 详见 [[居民入市6指标]] |
| **海外资金** | 北向 + QFII + 港资 | 此前连续 3 周净流入，本周转净流出但幅度收敛 |

## 框架的作用

- 用于判断"流动性牛市"叙事是否成立
- 任一蓄水池逻辑被打破 → 牛市叙事终结的信号
- 当前 9 月扰动多为**交易行为 + 情绪扰动**（月初事件落地后部分资金兑现 + 担忧监管降温），并非蓄水池逻辑挑战

## 核心结论

**指数赚钱效应 ↔ 增量资金 = 正螺旋**：
- 赚钱效应好 → 吸引增量资金 → 推升估值 → 强化赚钱效应

只要四个蓄水池"蓄势待发"的状态不被破坏，流动性牛市叙事就成立。
