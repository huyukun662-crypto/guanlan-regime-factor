# 噪声区域突破

> Noise Area Breakout — ETF 日内动量策略的核心信号机制

## 一句话定义

用过去 14 天**相同时间点的平均位移 σ**作为"正常波动"基准，结合昨收盘 / 今开盘价构造噪声区域上下边界。**价格突破上边界 → 多头需求；突破下边界 → 抛售压力**。

## 数学定义

```
move_{t-i, hh:mm} = | close_{t-i,hh:mm} / open_{t-i,9:30} - 1 |,    i = 1..14
σ_{t, hh:mm}     = (1/14) Σ move_{t-i, hh:mm}

UpperBound_{t, hh:mm} = max(open_{t,9:30}, close_{t-1,15:00}) × (1 + σ)
LowerBound_{t, hh:mm} = min(open_{t,9:30}, close_{t-1,15:00}) × (1 - σ)
```

## 信号规则

| 条件 | 信号 |
|---|---|
| close > UpperBound | 做多 |
| close < LowerBound | 做空 |
| LowerBound ≤ close ≤ UpperBound | 无 |

**判断时点**（避免边界震荡频繁信号）：仅在 10:29 / 11:29 / 13:59 三个 1 小时 K 线时点判断。

**平仓**：突破对向边界 → 即时平仓；当日收盘 → 强制平仓（不持仓过夜）。

## 关键改进

- **VWAP 止损线**：多头止损线 = max(UpperBound, VWAP)；空头 = min(LowerBound, VWAP)
- **结合昨收盘缺口**：UpperBound / LowerBound 用 max/min(开盘, 昨收) 而非单一开盘价

## 经济直觉

- 来自**粘性供需失衡**（sticky demand/supply imbalances）— Zarattini et al. 论文
- 价格对新信息**反应不足** → 一旦突破 14 天平均位移阈值，趋势可延续

## 与传统通道指标的对比

| 指标 | 通道宽度 | 时间维度 |
|---|---|---|
| **噪声区域（本指标）** | 过去 14 天**相同时点**位移均值 | 日内分钟级 |
| 布林带 | 过去 N 日标准差 | 日频 |
| ATR 通道 | True Range 均值 | 日频 |
| 唐奇安通道 | N 日 high/low | 日频 |

⇒ **核心差异**：噪声区域用"同时点"位移，**对日内时间结构敏感**（早盘 / 中盘 / 尾盘的波动差异被自动捕捉）。
