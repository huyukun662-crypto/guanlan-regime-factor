# 跨能源比价均值回归

> 原油 / 煤炭、原油 / 天然气等一次能源之间的**比价围绕中枢波动**——当某一比价偏离历史均值过远（如 >1σ），中长期有向中枢回归的倾向，构成跨商品相对价值（RV）信号。

## 简介

一次能源（原油、煤炭、天然气）中长期价格受全球能源需求、燃料替代、边际供给成本约束，比价通常围绕中枢波动。这是本库少见的**可直接量化的相对价值结构**（不同于纯叙事），可作为商品 / 能源板块内部的配置信号。

## 关键信息

- **类型**：概念 / 跨商品相对价值信号
- **领域**：大类资产、商品、能源板块择时
- **可量化形式**：`oil/coal`、`oil/gas` 比价的滚动 z-score；偏离 ±1σ 作为 RV 触发
- **数据源**：布伦特/WTI、动力煤、天然气（Henry Hub / TTF）期货价
- **相关概念**：[[OVX原油波动率指数]]、[[金铜比]]

## 详细内容

华泰宏观观察（2026/5）：中东冲突以来原油相对煤炭、天然气比价快速上升——**油价相对煤炭比价已高出近一个标准差，相对天然气也处历史偏高水平**。判断：
- 短期：若地缘扰动持续、霍尔木兹通行受限，原油相对其他能源比价仍可能进一步上行。
- 中长期：海峡复航、原油供给修复 + 风险溢价回落 → 比价向历史中枢回归，即油价相对承压。

## 与自有策略的接口

- 候选用法：油/煤、油/气比价 z-score > +1σ → 做空原油相对、或能源股内部从油气切向煤炭/电力；回归至中枢 → 平仓。
- 与 [[金铜比]] 思路同源（都是跨商品比价 regime），可纳入商品/资源板块的横截面打分。
- ⚠️ 比价中枢会随能源结构转型（电动化、天然气替代）缓慢漂移，需用滚动窗口而非固定历史均值；原文无回测。**仅供研究参考。**
