# 低延时趋势

> 浙商证券[[多信号策略]]中的趋势识别核心模块 — 通过**频域滤波**克服传统均线滞后性

## 一句话定义

时域中识别价格波动困难 → **变换到频域**（拉普拉斯 / 傅里叶变换）→ **过滤高频成分**保留低频部分 → 再变回时域得到"低延时趋势曲线"。算法用**平滑系数 α**控制响应速度与平稳性的权衡。

## 与传统均线的对比

| 维度 | 传统 EMA / MACD | **低延时趋势（本策略）** |
|---|---|---|
| 数学基础 | 指数加权平均 | 信号处理 / 频域滤波 |
| 滞后性 | **明显**（趋势中段才入场） | **大幅降低**（快速定位趋势起点） |
| 响应速度 | 慢 | 快 |
| 平稳性 | 较好 | 通过 α 系数可调 |
| 实施复杂度 | 简单（标准库） | 中等（需 scipy.signal） |

## 信号生成机制

1. 用拉普拉斯 / 傅里叶变换将价格序列转换为复频域
2. 在算法内部通过 α 参数控制响应速度 vs 平稳性
3. 将低频部分变回时域得到"低延时曲线"
4. 计算曲线在观察窗口（5-30 日）内的**平均斜率**
5. 若 |斜率| > 阈值（如 1 或 0）→ 判定为有效趋势 → 开仓做多
6. 通过杠杆参数调节策略放大倍数

## 优势

- **趋势起点识别** — 胜率与回撤控制优于传统 EMA / MACD
- **超预期事件触发后快速定位趋势起点** — 显著提升入场效率

## 劣势

- 弱趋势 / 横盘震荡 → 反复建仓信号 → **需配合其他模块过滤震荡区间**

## 与同主题指标的关系

| 指标 | 数学基础 | 滞后性 |
|---|---|---|
| MACD | EMA 双线差 | 高 |
| **低延时趋势** | 频域滤波 | **低** |
| [[均线偏离度]] | ln(close) - ln(EMA20) | 中 |
| [[阶矩择时]] | 高阶统计矩 | 中（前瞻性） |

## 在动量衰竭早期识别框架中的角色

低延时趋势是 [[动量衰竭早期识别]] 三层框架中的**趋势导数层**（中间层）：
- 入场用：低延时斜率由负转正 + 持续 K 日 → 确认趋势起点
- 止盈用：低延时斜率由正转平 / 由正转负 → **趋势开始失速** → 减仓信号
- 比起九转的离散计数，低延时给出连续斜率值，可分级触发减仓（斜率拉平 30% 减仓 / 转负 50% 减仓）
- 比起 [[VPIN]] 的微观结构信号，低延时仅用收盘价但更滞后，作为 VPIN 不可得场景下的"次优替代"
