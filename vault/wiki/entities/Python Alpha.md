# Python Alpha

## 定义

BRAIN 平台 2026 新功能，允许用 Python 代码（可导入 NumPy/Pandas 等库）直接替换表达式做回测，复用 simulation API。

## 核心要点

- 优势：策略相关性更低，适合挖掘新信号
- 摆脱原 BRAIN 表达式 DSL 的语法限制
- 复用：现有 simulation API、回测基础设施
- 相关：[[BRAIN Python API]] [[WorldQuant BRAIN平台]]
