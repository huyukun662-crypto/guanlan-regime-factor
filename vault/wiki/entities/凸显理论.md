# 凸显理论（Salience Theory）

> Bordalo-Gennaioli-Shleifer 2012 提出的行为金融学理论 — **投资者注意力被截面凸显的极端结果吸引**

## 核心命题

资产横向比较中：
- 投资者**注意力被"凸显"（salient）的回报吸引**
- **不凸显的回报被忽略**
- 凸显程度由"在截面上偏离平均的程度"决定

具体来说：
- 某只股票当月**涨停** → 凸显 → 被记住
- 某只股票当月微涨 2% → 不凸显 → 被忽略

⇒ 投资者的决策权重不等于客观概率，而是被**凸显性扭曲**。

## 与前景理论的对比（关键区别）

| 维度 | [[前景理论]] | **凸显理论（本理论）** |
|---|---|---|
| 提出者 | Kahneman-Tversky 1979 | BGS 2012 |
| 极端收益获更高权重的原因 | **概率小**（小概率事件被高估）| **截面凸显**（相对市场平均偏离大）|
| 信息维度 | 时序（单股票）| **时序 + 截面** |
| 应用因子 | TK 因子（Barberis 2016）| **STR 因子**（Cosemans-Frehen 2021）|

凸显理论的优势：
- **同时含时序 + 截面信息**（前景理论只有时序）
- 解释了"个股相对市场涨跌"为何被过度反应

## 核心数学（凸显度 σ）

```
σ(r_{i,d}) = |r_{i,d} - r̄_d| / (|r_{i,d}| + |r̄_d| + 0.1)
```
- 分子：偏离平均的绝对距离
- 分母：偏离 + 自身基准（防 0 项 0.1）
- 输出：[0, 1] 之间，越大越凸显

## ST > 0 / ST < 0 的市场含义

### ST > 0：上行凸显
- 股票最高回报突出
- 投资者过度关注上行潜力
- 风险寻求 → 过度买入
- **未来下跌**（反向因子）

### ST < 0：下行凸显
- 股票最低回报突出
- 投资者强调下行风险
- 过度低估 → 过度卖出
- **未来上涨**（反向因子）

⇒ STR 因子和很多行为金融因子一样是**反向因子**（多空头都做反方向）。
