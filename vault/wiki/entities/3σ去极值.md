# 3σ 去极值

> Winsorization 的标准做法：将偏离均值 3 倍标准差以外的值截尾

## 公式

```
upper = mean(x) + 3 × std(x)
lower = mean(x) - 3 × std(x)
x' = clip(x, lower, upper)
```

## 适用场景

- 因子值分布存在长尾或极端异常值
- 模型对极端值敏感（线性回归、神经网络）
- 标准化前的预处理

## 与其他去极值方法对比

| 方法 | 原理 | 鲁棒性 |
|---|---|---|
| **3σ 法（本方法）** | 均值 ± 3 倍标准差 | 中（受极端值影响） |
| MAD 法 | 中位数 ± k × MAD | 高（基于中位数） |
| 分位数法 | 取 [1%, 99%] 分位 | 高 |
