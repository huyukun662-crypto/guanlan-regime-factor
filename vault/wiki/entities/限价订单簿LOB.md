# 限价订单簿 (LOB)

> Limit Order Book — 全球交易所的核心匹配机制 — 高频交易的"地图"

## 一句话定义

LOB 是**匿名记录买卖双方限价指令**的电子簿，按价格水平分层（每层间隔 = 最小价格增量），交易所按"价格优先 + 时间优先"撮合。**是高频交易、做市、最优执行的所有研究的基础数据结构**。

## 基本结构

```
卖方（Ask 卖一/卖二/卖三 ...）
─────────────────────
$10.05  ← Ask3：500 股
$10.04  ← Ask2：1200 股
$10.03  ← Ask1：800 股   ← 最优卖价
─────────────────────  ← 价差 = $0.02
$10.01  ← Bid1：600 股   ← 最优买价
$10.00  ← Bid2：900 股
$9.99   ← Bid3：1500 股
─────────────────────
买方（Bid 买一/买二/买三 ...）
```

## 两类指令

| 指令类型 | 行为 | 影响 |
|---|---|---|
| **限价指令**（Limit Order）| 加入 LOB 等待匹配 | 增加流动性（提供 maker 角色） |
| **市场指令**（Market Order）| 立即在最优反向价成交 | 消耗流动性（taker 角色） |

**最小价格增量**：
- 美股：$0.01（多数股票）
- 期货：取决于合约名义价值

## 关键观测指标（来自第 5 章）

### 1. 交易量不平衡（Volume Imbalance）— **关键预测信号**
```
imbalance = (bid_volume - ask_volume) / (bid_volume + ask_volume)
```
- 在 [+1, -1] 之间
- 正值 → 买方力量强 → 短期价格上行概率增加
- 负值 → 卖方力量强 → 短期价格下行概率增加

### 2. 价差（Spread）
- 最优卖价 - 最优买价
- 流动性的直接度量

### 3. 订单到达率（Arrival Rates）
- 限价单到达率 / 市场单到达率 / 撤单率
- 用 [[Hawkes过程]]（自激点过程）建模（来自第 5.3 节）

### 4. 订单深度（Depth）
- 各层级累积量
- 反映抗冲击能力

## 三类建模方法（来自本书 5.3-5.5）

### 1. 多元点过程模型（5.3）
- 用 [[Hawkes过程]] / 复合 Poisson 过程拟合订单到达
- 捕捉**订单聚簇**（一笔大单触发跟随单）
- 适合**短期价格方向预测**

### 2. 机器学习（5.4）
- 输入：LOB 多层级特征 + 订单流
- 输出：未来短期价格方向 / 价差变化
- 现代主流方向（深度学习兴起后被广泛应用）

### 3. 排队论（5.5）
- 把每个价格层级看作 M/M/1 队列
- 扩散极限 / 流动极限
- 适合理论分析（撮合速度、价差稳态）

## 与已有素材的关联

### 与 [[2026-05-05-qpb-etf-intraday-momentum]]（西部 ETF 日内动量）
- ETF 日内动量用**分钟 K 线**（已是 LOB 聚合的低分辨率版本）
- LOB 提供**毫秒级**原始数据 → 信号源更精细
- 本书第 6 章基于 LOB 数据做最优执行 → 与 ETF 日内动量是同一问题不同分辨率

### 与 [[多信号策略]]（浙商证券 2025）
- 浙商系列二的"通道过滤"是**日频 K 线**层面
- LOB 通道过滤可在毫秒级实现 → 高频版本

### 与 V25 项目
- V25 目前是**日频 ETF**，未触及 LOB
- 如要扩展到分钟 / tick 级 → LOB 是必经之路

## 数据获取难度

| 数据类型 | 颗粒度 | 来源 | 成本 |
|---|---|---|---|
| LOB 全档 | 毫秒级 | 交易所 / 数据商（Wind / 朝阳永续 / Tickdata）| **极高**（专业数据商）|
| LOB Top-5 | 秒级 | 部分券商 API | 中（需开户）|
| 分钟 K 线（汇总）| 1 分钟 | [[akshare]] / efinance | **低**（公开免费）|
| 日 K 线 | 日 | 公开 | 极低 |

⇒ V25 当前层级不需要 LOB，但**未来扩展高频时必须解决数据问题**。

## 优势 / 劣势

✅ **信息最丰富**的市场数据形式
✅ 是所有高频策略的"原始素材"
✅ 排队论 / 点过程等经典数学工具可建模

⚠ 数据成本极高
⚠ 算法复杂度高（需要专业基础设施）
⚠ 实盘延迟敏感（毫秒级争夺）
⚠ A 股 LOB 数据公开度比美股低
