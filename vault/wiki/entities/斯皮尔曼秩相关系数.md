# 斯皮尔曼秩相关系数

> Spearman Rank Correlation — 抗极端值的稳健相关性度量

## 一句话定义

将原始值替换为**排名（rank）**后计算 Pearson 相关系数。**对极端值不敏感**，是量化金融中计算 IC（信息系数）的首选方法。

## 公式

```
ρ_s = Pearson(rank(x), rank(y))
   = 1 - 6·Σd_i² / (n·(n²-1))    # d_i = rank(x_i) - rank(y_i)
```

## 与 Pearson 相关系数对比

| 维度 | Pearson | **Spearman（本方法）** |
|---|---|---|
| 关系假设 | 线性 | 单调（包括非线性单调） |
| 抗极端值 | 弱 | **强** |
| 量化用途 | 因子线性合成 | **IC / ICIR 计算** |

## 量化金融中的标准用法

```python
from scipy.stats import spearmanr

# 单期 IC
ic, p_value = spearmanr(factor_values_t, future_returns_t1)

# 多期 IC 时间序列 → ICIR
ic_series = []
for t in range(T):
    ic, _ = spearmanr(factors[t], returns[t+1])
    ic_series.append(ic)
icir = np.mean(ic_series) / np.std(ic_series)
```

## 为什么量化用 Spearman 而不是 Pearson？

- 收益率分布有重尾，单只股票极端值（涨停 / 跌停 / 一字板）扰动 Pearson
- 因子值也常有长尾，原始值乘积易被少数样本主导
- 排名后 → 每个样本贡献相等，更反映"整体方向性"
