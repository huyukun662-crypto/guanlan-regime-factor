# NH-NL 指标（净新高占比）

> Net Percent of New High Minus New Low — 行业指数顶底情绪信号（华福证券 2023）

## 一句话定义

```
(NH-NL)% = (创 52 周新高个股数 - 创 52 周新低个股数) / 行业内上市超 1 年个股数
```

用行业内**新高新低之差占比**衡量行业指数情绪强弱，5 档分类（贪婪 / 乐观 / 正常 / 悲观 / 恐惧），可作为反转策略信号。

## 阈值（5 档）

| (NH-NL)% | 情绪 |
|---|---|
| ≥ 30% | **贪婪** |
| 20%~30% | 乐观 |
| -20%~20% | 正常 |
| -30%~-20% | 悲观 |
| ≤ -30% | **恐惧** |

小行业（< 40 只）阈值放宽为 ±30% / ±40%。

## 关键细节（与传统不同）

1. 用 close 而非 high/low — 收盘价更稳定
2. 识别窗口为 T_{1-52}（不含今日）— 避免今日实时跳动
3. 上市 ≥ 1 年的个股才计入分母 — 排除新股噪音
4. 归一化 [-1, 1] → **可跨行业横向比较**（去除行业规模差异）

## 经济直觉（行为金融）

- **锚定效应**：投资者记得成本价 → 浮盈惜售 / 浮亏即抛
- 创新高个股 → 持有者均浮盈 → 抛压小
- 创新低个股 → 持有者均浮亏 → 反弹即抛

## 适用场景

✅ 行业级择时（科技 / 周期表现最好）
✅ 反转策略（贪婪做空 / 恐惧做多）
✅ 作为指数增强信号（资金占用时间短）
⚠ 不适合趋势跟随（本质是反转）
⚠ 行业成分股调整影响（半年一调）
