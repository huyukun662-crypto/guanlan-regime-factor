# 价值投资量化框架

> "先定行业壁垒，再选优质标的"两层量化筛选框架

## 一句话定义

本框架基于**价值投资经典理论**（企业价值 = 资产价值 + 盈利能力价值 + 成长价值），将价值投资的"竞争壁垒+优质标的"逻辑落地为**行业分类 + 标的筛选**两层量化流水线，转向**低换手、高胜率、高赔率的高精度标的筛选**。

## 解决的传统价投三大局限

1. **识别企业竞争壁垒依赖深度行业研究**，普通投资者难以复现
2. **估值判断主观性强**，难以形成稳定的投资纪律
3. **持仓周期长**，对短期市场波动的容忍度要求高，资金效率受限

## 解决的传统因子投资问题

- 传统因子投资 = 大数定律 + 广撒网 → 高换手 / 低胜率 / 风格切换易失效
- 本框架 = 高精度筛选 → 低换手 + 高胜率 + 高赔率

## 框架结构

### 第一层：行业分类

用两个核心指标划分**高壁垒稳态行业** vs **充分竞争行业**：

#### [[行业格局稳态性指标]] S
```
D_{k,t} = ||V_{k,t} - V_{k,t-1}||₂          # 市值向量欧氏距离
D̄_{k,T} = median(D_{k,T-11..T})             # 12 期中位数
S_{k,T} = 1 - percentile(D̄_{k,T})           # 截面百分位变换
```

#### [[行业盈利可持续性指标]] P
```
ROĒ_{k,T} = median(ROE_{k,T-11..T})         # 12 期 ROE 中位数
P_{k,T}   = percentile(ROĒ_{k,T})           # 截面百分位
```

#### 阈值
- **高壁垒稳态行业**：S > 0.7 且 P > 0.7
- **充分竞争行业**：其余

### 第二层：标的筛选

#### 高壁垒行业 → [[寡头共赢标的]]
若不存在绝对龙头，几家势均力敌的寡头企业更易合作共赢、避免价格战、长期高盈利。

筛选条件：
1. 先决：高壁垒稳态行业 + 无绝对龙头
2. **盈利质量**：滚动 3 年调整后营业利润率中位数前 30%
3. **现金流质量**：滚动 3 年经营活动净现金流/营业利润中位数前 30%
4. 终选：等权综合打分前 1 / 行业

#### 充分竞争行业 → [[高效运营组合]]
通过成本控制 + 资产周转效率获得超额。

筛选条件：
1. 先决：充分竞争行业 + 行业内成分股 ≥ 10 只
2. **成本控制能力**：滚动 3 年销售/管理/财务费用率中位数前 30%
3. **运营效率**：滚动 3 年流动资产周转率/存货周转率/应收账款周转率/ROIC 前 30%
4. 终选：等权综合打分前 1 / 行业；总数 > 60 取前 60 控制容量

## 优化方向（作者自述）

1. 加入估值过滤模块（PB-ROE 估值匹配度 → 规避高估值标的）
2. 优化行业壁垒识别（专利数据、品牌力、牌照资源等另类数据）
3. 加入仓位管理模块（按市场波动率、宏观经济周期调整组合权重）
4. 加入流动性指标（适配大资金实盘配置）
